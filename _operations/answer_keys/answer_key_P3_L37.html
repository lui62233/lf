P3 下學期SSPA模擬 參考答案
LF-P3-下-L37 · 對稱初探 · 軸對稱與對稱軸 · 65分鐘
僅供教師及家長核對使用。學生必須自行完成後方可查閱。
一、熱身啟動題 (Q1-Q5)
Q1. 對稱圖形摺埋之後，兩邊會＿＿＿＿（一模一樣 / 唔同）。
答案：一模一樣
軸對稱的定義：沿對稱軸對摺後，兩邊完全重合。
Q2. 一個心形左右兩半係咪對稱？
答案：係，心形係左右對稱（1條垂直對稱軸）
心形沿中間垂直線對摺，左右兩邊會完全重合。
Q3. 正方形有幾多條對稱軸？A. 1條 B. 2條 C. 4條
答案：C. 4條
正方形有4條對稱軸：2條穿過對邊中點（垂直+水平），2條穿過對角線。這是P3-P4最高頻陷阱！
Trap T5 (幾何圖形混淆)：最常見錯誤是選B (2條)，把正方形當成長方形。正方形同時具備長方形和菱形的對稱軸！
Q4. 長方形有幾多條對稱軸？A. 1條 B. 2條 C. 4條
答案：B. 2條
長方形只有2條對稱軸：穿過上下邊中點的水平線 + 穿過左右邊中點的垂直線。
對角線唔係對稱軸！（沿對角線摺，兩邊唔會重合）
Trap T5 (幾何圖形混淆)：最常見錯誤是選C (4條)，以為長方形同正方形一樣有4條。長方形對角線不是對稱軸！另一錯誤是選A (1條)，忘記了可以有水平和垂直兩條。
Q5. 圓形有幾多條對稱軸？
答案：無限多條
任何穿過圓心的直線都是圓形的對稱軸。因為圓形在所有方向上都對稱，所以有無限多條。
Trap T5 (幾何圖形混淆)：常見錯誤是答「0條」、「2條」或「4條」。圓形係最特別既圖形！
二、知識點一 同步練習 (Q6-Q8)
Q6. 以下邊啲係軸對稱圖形？A. 圓形 B. 不等邊三角形 C. 等腰三角形 D. 正方形
答案：A, C, D
A. 圓形 → 係（無限多條對稱軸）
B. 不等邊三角形 → 唔係（三條邊長度都唔同，冇對稱軸）
C. 等腰三角形 → 係（1條對稱軸，由頂點到底邊中點）
D. 正方形 → 係（4條對稱軸）
Q7. 英文字母「A」有對稱軸嗎？「B」呢？「C」呢？
答案：A有（1條垂直）、B有（1條水平）、C有（1條水平）
A：沿中間垂直線對摺，左右對稱 → 1條
B：沿中間水平線對摺，上下對稱 → 1條
C：沿中間水平線對摺，上下對稱 → 1條
Trap T5：注意對稱軸方向可以係垂直（如A）、水平（如B、C），唔一定係垂直！
Q8. 一個等邊三角形有幾多條對稱軸？
答案：3條
等邊三角形有3條對稱軸，每條由一個頂點畫到對邊的中點。三條邊一樣長，三個角一樣大，所以三個方向都對稱。
Trap T5：不要和等腰三角形（只有1條）混淆！等邊三角形係最對稱既三角形。
三、知識點二 同步練習 (Q9-Q12)
Q9. 正方形有幾多條對稱軸？畫出全部。
答案：4條
兩條穿過對邊中點（垂直中線+水平中線），兩條穿過對角線（左上至右下+右上至左下）。
Q10. 等腰三角形有幾多條對稱軸？不等邊三角形呢？
答案：等腰三角形有1條；不等邊三角形有0條
等腰三角形：唯一對稱軸由頂點（兩條相等邊的交點）垂直畫到底邊的中點。
不等邊三角形：三邊長度都唔同，冇任何對稱軸。
Trap T5：唔係所有三角形都有對稱軸！只有等腰和等邊三角形先有。
Q11. 在方格紙上（6x6格），畫一個對稱圖形（左右對稱）。
答案：畫出一個左右對稱的圖形（例如心形、蝴蝶形、箭頭形）
評分要點：(1) 有標示對稱軸 (2) 左邊和右邊是鏡像 (3) 對應點到對稱軸的距離相同。
常見例子：在中間垂直線兩側各畫一個對應的方格圖案。
Q12. 正五邊形有幾多條對稱軸？
答案：5條
正n邊形有n條對稱軸。正五邊形有5條：每條由一個頂點穿過對邊的中點。
規律：正多邊形的對稱軸數量 = 邊的數量。
Trap T5：不要猜成3條或4條。記住規則：正n邊形有n條對稱軸。
四、課堂分層同步練習 (Q13-Q21)
Q13. 對稱軸係一條＿＿線，圖形沿佢對摺後兩邊會＿＿。
答案：直線；重合（或「一模一樣」）
Q14. 正方形有＿＿條對稱軸。
答案：4
Q15. 以下邊個數字有對稱軸？0, 1, 2, 3, 8（提示：諗下沿橫線摺）
答案：0和8有水平對稱軸；3沒有標準對稱軸
0：沿水平線摺，上下對稱（有水平對稱軸）
8：沿水平線摺，上下對稱（有水平對稱軸）
1, 2, 3：沒有對稱軸
Q16. 一個正六邊形有幾多條對稱軸？
答案：6條
3條穿過對角（相對的頂點）+ 3條穿過對邊中點（相對的邊的中點）。總共6條。
Q17. 長方形既對角線係咪對稱軸？用摺紙法解釋！
答案：唔係！
沿長方形對角線摺，兩邊唔會重合（除非係正方形）。因為長方形的長和闊唔同，沿對角線摺時，兩邊的大小和形狀唔同。正方形既對角線先係對稱軸。
Trap T5 (幾何圖形混淆)：這是P3-P6遇到頻率最高的陷阱之一！好多人以為長方形對角線係對稱軸。
Q18. 在方格紙上畫一個「十」字形。呢個圖形有幾多條對稱軸？
答案：4條
十字形有4條對稱軸：1條垂直（穿過中間）、1條水平（穿過中間）、2條45度對角線（如果十字的四臂長度相同）。如果十字臂長度唔同，則只有2條。
Q19. 一個菱形（四條邊相等但唔係正方形）有幾多條對稱軸？
答案：2條（兩條對角線）
菱形（非正方形）：四邊相等但角不是90度。對稱軸 = 兩條對角線。沿對角線摺，兩邊會重合。
如果菱形是正方形，就有4條（再加兩條中線）。
Trap T5：菱形和正方形的分別在於角度。菱形有2條對稱軸（對角線），正方形有4條（對角線+中線）。
Q20. 在方格紙上，完成一個左右對稱的圖案：左半邊已畫，請畫出右半邊的鏡像。
答案：右半邊是左半邊的鏡像，每個對應點到對稱軸的距離相同
畫法：對於左邊每一個著色的方格，在右邊相同距離處畫上對應方格。對稱軸是中間的垂直線。
Q21. 摺紙挑戰：對摺一次，剪一刀。打開後得到咩形狀？對摺兩次（十字摺），再剪一刀呢？
答案：(1) 對摺一次剪一刀 → 對稱圖形，有1條對稱軸
(2) 對摺兩次（十字摺）剪一刀 → 有4條對稱軸的圖形（類似雪花）
每次對摺會增加對稱軸的數量。對摺n次，最多可以有2^n個重複圖案。
五、生活應用題 (Q22-Q26)
Q22. 搵出屋企3樣對稱既物品。
答案示例：碟（圓形，無限對稱軸）、書（長方形，2條）、杯（圓柱形截面，無限）
Q23. 寫出5個有對稱軸既大寫英文字母。
答案示例：A（垂直1條）、H（垂直+水平2條）、I（垂直+水平2條）、M（垂直1條）、O（無限條）
其他有對稱軸的字母：T, U, V, W, X, Y (垂直)；B, C, D, E, K (水平)
Q24. 企喺鏡前面：你的左邊喺鏡入面變成右邊。呢個係咪對稱？對稱軸喺邊？
答案：係，這是鏡像對稱。對稱軸在鏡面位置。
鏡像對稱（reflection symmetry）：物體和鏡中的影像關於鏡面對稱。鏡面就是對稱軸所在的平面。
Q25. 用摺紙+剪刀，創作出一個對稱圖案。
答案：實際操作題，無固定答案。評分看對稱性和創意。
Q26. 觀察建築物：搵一個左右對稱的建築物例子，畫出外形，標示對稱軸。
答案示例：故宮、泰姬陵、教堂正面，均為左右對稱。
六、課後功課 (H1-H8)
H1. 圓形有幾多條對稱軸？A. 1條 B. 2條 C. 無限多
答案：C. 無限多
H2. 正方形和長方形，邊個對稱軸多啲？多幾多？
答案：正方形多啲。正方形有4條，長方形有2條。正方形比長方形多2條。
H3. 人體（外表）大約有幾多條對稱軸？係咩方向？
答案：1條，垂直方向（左右對稱）
H4. 以下圖形有冇對稱軸？如有，有幾多條？A. 等腰梯形 B. 平行四邊形
答案：A. 等腰梯形有1條（垂直對稱軸）
B. 平行四邊形有0條（一般平行四邊形没有對稱軸）
Trap T5：平行四邊形（非特殊）冇對稱軸！呢個係P3-P6常見陷阱。
H5. 摺紙實驗：對摺一張紙一次，剪一個半心形。打開後係咩形狀？有幾多條對稱軸？
答案：打開後係一個完整的心形。有1條對稱軸（摺痕位置）。
H6. 正八邊形（stop sign形狀）有幾多條對稱軸？
答案：8條
正n邊形有n條對稱軸。4條穿過對角 + 4條穿過對邊中點。
H7. 在方格紙上設計一個有2條對稱軸的圖案（上下對稱+左右對稱）。
答案：需要同時具備水平和垂直對稱軸。例如：正方形圖案、十字形、四葉草形。
H8. 雪花通常有6條對稱軸。你能用摺紙方法（對摺3次）剪出一個六角對稱既雪花嗎？
答案：將正方形紙對摺成60度角的扇形（需要摺出6等份），再剪圖案。打開後可得6角對稱雪花。
