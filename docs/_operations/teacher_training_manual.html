呢本手冊嘅目的：將你由一個「識計數嘅人」變成一個「識睇穿錯誤嘅人」。

每一個霖楓學生，上完一堂之後，記住嘅唔係公式，而係：「我今日差啲中咗T4陷阱，好在我識睇。」

霖楓學苑嘅產品唔係講義。講義係道具。你，先係真正嘅產品。你呢65分鐘嘅表現，直接決定一個學生留低定走、進步定原地踏步、鍾意數學定驚咗數學。

陷阱為先教學法（Trap-First Pedagogy）

傳統補習嘅流程：解釋概念 → 示範例題 → 學生練習 → 改正。呢個流程有一個致命缺陷：佢假設學生聽完正確方法就會記得。但現實係——學生返到考試現場，第一時間浮現嘅唔係你教嘅正確方法，而係佢一直以嚟嘅錯誤直覺。

🔴 關鍵數據：研究顯示，學生喺壓力下（考試），有 67% 機會率先提取最熟悉嘅路徑——即使嗰條路徑係錯嘅。呢個就係點解「教啱嘢」唔夠，你必須「拆除錯嘢」。

霖楓學苑嘅教學法將流程倒轉：

🪤 先展示錯誤
「睇下——呢個答案係錯嘅。點解60%學生會咁計？」
→
💡 揭露陷阱機制
「因為佢哋直接乘咗——但呢度有個隱藏嘅÷2！」
→
🧠 強化正確記憶
「所以我每次見到三角形面積，第一件事係——」

呢個就係「陷阱引爆」：每一堂嘅每一個核心知識點（KP），你必須包含最少一個陷阱引爆時刻。展示錯誤答案 → 叫學生指出錯喺邊 → 然後先揭示正確解法。呢個情緒起伏（「好彩我冇中招！」）係最強嘅記憶黏合劑。

🧠 神經科學基礎：當人發現自己「差啲犯錯」嘅時候，大腦會釋放去甲腎上腺素——呢個化學物質直接增強海馬體嘅記憶編碼。簡單講：「差啲中伏」嘅情緒 = 永久記憶。所以你唔係喺度嚇學生，你係喺度幫佢嘅大腦將正確方法「焊死」入長期記憶。

65分鐘情緒弧線（The Emotional Arc）

一堂霖楓課堂唔係線性嘅知識灌輸。佢係一條精心設計嘅情緒曲線：

每一個階段有明確嘅情緒目標：

失敗階段（0-20min）：學生必須經歷「發現自己唔識」嘅不適感。呢個不適感係必要嘅——冇佢，後面嘅洞察就冇衝擊力。你呢個階段嘅角色係「安全網」：俾學生犯錯，但唔好俾佢覺得自己蠢。

洞察階段（20-45min）：呢個係課堂嘅高峰。學生由「我唔識」變成「我睇穿咗！」。你呢個階段嘅角色係「導遊」：帶學生穿越陷阱區，每穿越一個就慶祝一次。

記憶階段（45-65min）：將洞察轉化為自動反應。口訣、應用題、出門券。你呢個階段嘅角色係「教練」：確保肌肉記憶形成。

🧠 口訣：「失敗係燃料，洞察係引擎，記憶係目的地。」

導師角色：陷阱診斷師，不是知識傳遞員

傳統老師嘅自我定位：「我要將呢課書教晒。」
霖楓導師嘅自我定位：「我要確保學生離開課室之前，識得避開呢課嘅所有致命陷阱。」

兩者嘅分別在於：前者以「內容」為中心，後者以「錯誤」為中心。

傳統老師霖楓導師（你）
「呢課要教XXX」 「呢課嘅3個致命陷阱係XXX，我要確保學生全部識避」
見到學生錯 → 改正答案 見到學生錯 → 診斷陷阱類型 → 引爆陷阱 → 強化正確路徑
「你計錯咗，正確答案係...」 「你啱啱中咗T4陷阱！睇下——你係咪漏咗÷2？」
學生識做 = 教完 學生識做 + 識講得出點解其他人會錯 = 教完
講義 = 教學內容 講義 = 陷阱觸發裝置
🎯 你嘅核心任務得一個：每一堂，每一個學生離開課室嘅時候，能夠講出一句：「今日我學識避開______陷阱。」如果佢講唔出，你呢堂未完成。

三軌系統：🚀Fast / 🚶Standard / 🐢Support

1-on-3 課堂嘅最大挑戰：三個學生嘅能力差距可以好大。一個已經識晒，一個跟到，一個完全跟唔上。三軌系統嘅目的唔係標籤學生，而係確保每個學生喺自己嘅「挑戰區」入面學習。

軌道學生特徵你點樣處理練習難度
🚀 Fast KP概念15秒內理解；基礎題全對；會主動問「如果...會點？」 俾佢跳過基礎練習，直接進入🌳/🏔️級別；要求佢用語言解釋陷阱俾🚶學生聽 🌳 → 🏔️
🚶 Standard 需要睇一次完整示範先做到；基礎題7-8成準確率 標準流程；每個KP一個陷阱引爆；做完🌱先升🌿 🌱 → 🌿 → 🌳
🐢 Support 需要2-3次示範；基礎題低於5成準確率；可能缺乏前置知識 縮減練習量，聚焦1-2個最關鍵陷阱；大量正面肯定；用「錯得有價值」框架 🌱（只做基礎）
⚠️ 分軌關鍵規則：分軌只發生喺30-45min嘅「分層練習」時段。其餘時間（熱身、KP教學、應用題、出門券），三個學生一齊上堂。唔好俾學生感覺自己被「降級」——🌱🌿🌳🏔️係題目難度標記，唔係學生能力標籤。

作為霖楓導師，你嘅每一句說話都係工具。以下係你必須內化嘅核心語言：

場景標準用語禁用用語
學生答錯 「好嘢！你啱啱中咗T3陷阱。呢個陷阱考起60%嘅學生。我哋一齊睇下佢點樣運作。」 「錯咗。」「唔啱。」「再諗下。」

0-5min：熱身啟動（Warm-Up Activation）

目標：30秒內建立「呢堂好重要」嘅感覺。激活上一堂嘅知識（唔係呢堂嘅新嘢）。

0:00-0:30
入場儀式
「各位，今日我哋要面對嘅係SSPA其中一個最致命嘅陷阱區——（停頓）——面積公式混淆。近5年SSPA，每次都有最少3分係因為呢個陷阱區扣走嘅。3分 = 一個Grade。你準備好征服佢未？」
0:30-3:00
熱身題（教師示範，學生抄寫）
喺白板/屏幕上展示1-2題上一堂嘅陷阱題。教師邊講邊寫完整步驟。學生喺自己講義上抄低。目的：喚醒上一堂嘅陷阱記憶，同時俾遲到嘅學生有緩衝時間。
關鍵：教師一路寫一路自言自語式講出思考過程。「呢度我見到三角形——三角形面積 = 底×高÷2。有人會漏咗÷2——嗰個就係T4陷阱。」
3:00-5:00
今日課題預告 + 陷阱預警
展示今日課題名稱。用一個「世紀災難級」錯誤例子吸引注意。例如：「小明計梯形面積嘅時候——用咗平行四邊形公式。結果成條長答8分全部扣晒。今日，你唔會犯呢個錯誤。」然後揭示今日會學嘅3個KP同對應嘅陷阱。

5-20min：核心知識精講（KP Teaching）

結構：每堂3個KP，每個KP嚴格5分鐘。每個KP包含：概念（1min）→ 陷阱展示（1min）→ 陷阱引爆（1min）→ 正確方法（1min）→ 學生複述（1min）。

KP1
5:00-10:00
第一個核心知識點
0-1min：快速講解概念（唔好超過60秒）。「梯形面積 = (上底+下底)×高÷2。記住——係兩條底嘅平均，乘高。」
1-2min：陷阱展示。展示一個故意犯錯嘅計算。「睇下小明點計：（展示錯誤答案）。佢做咗咩？」
2-3min：陷阱引爆。叫一個學生指出錯誤。「小明漏咗——？」等學生答「÷2！」然後你大聲確認：「冇錯！T4陷阱！梯形面積一定要÷2！」
3-4min：正確示範。完整展示正確步驟，每個步驟標註陷阱檢查點。
4-5min：學生複述。叫一個學生用自己文字解釋陷阱同正確方法。
KP2
10:00-15:00
第二個核心知識點
同上結構。KP2通常係KP1嘅延伸或對比。例如KP1係梯形面積，KP2可以係菱形面積（另一種÷2情況，但原理不同）。重點：KP2嘅陷阱必須同KP1嘅陷阱有「對比」效果——等學生明白「÷2」喺唔同圖形入面嘅唔同原因。
KP3
15:00-20:00
第三個核心知識點
同上結構。KP3通常係最難嘅概念，或者係綜合應用。例如：複合圖形面積（需要先分解成基本圖形，每個各自用公式，再相加）。呢度嘅陷阱通常係T5（幾何圖形混淆）。
⏱️ 時間紀律提示：每個KP一到5分鐘即刻停，即使學生未完全掌握。記住：後面仲有35分鐘練習時間。KP精講嘅目的係「展示地圖」，唔係「行完全程」。

20-30min：輪流實戰（Round-Robin Practice）

每個學生輪流做1題，大聲講出思考過程。其餘2個學生嘅任務唔係等——係「檢查」：佢哋要判斷做緊嗰個學生有冇中陷阱。

流程：學生A做題出聲 → 學生B同C喺自己講義上寫低「佢有冇中陷阱？邊個？」→ 教師確認診斷 → 輪到下一個學生。10分鐘，每人約3分鐘。

30-45min：分層練習（Tiered Practice）

呢個係課堂最關鍵嘅15分鐘。三個學生做唔同難度嘅練習。你嘅注意力分配：50% 🐢Support → 30% 🚶Standard → 20% 🚀Fast。

🚀Fast：俾佢做🌳級別題目（綜合陷阱題）。如果做完，叫佢設計一題陷阱題考你。
🚶Standard：由🌱做起，升🌿。每做完一題俾你檢查先做下一題。
🐢Support：坐喺你隔離。你帶住佢一題一題做。每一步都問：「呢度有冇陷阱？」

45-55min：應用題+挑戰

全班一齊。文字題/情境題，包含跨陷阱綜合挑戰。教師讀題 → 全班一齊拆解 → 各自完成 → 對答案。呢個時段展示「真實考試會點樣混合陷阱」。

55-60min：出門券（Exit Ticket）

🔴 鐵律：出門券 = 1題。必須涵蓋今日最關鍵嘅陷阱。做啱 → 落堂。做錯 → 你即場引導佢發現自己嘅錯誤，再做一次類似題目，直到做啱為止。呢個唔係懲罰，係確保「記憶焊接」完成。

60-65min：功課佈置+口訣背誦

功課：3-5題針對性陷阱題（唔係大量操練）。指明每題對應邊個陷阱。
口訣背誦：全班一齊大聲背誦今日口訣2-3次。最後一次由最快嗰個學生帶讀。

Part 3 · 即時陷阱診斷

診斷篇 — 5秒內辨認學生中咗邊個陷阱
導師培訓手冊
第3部
🎯 診斷能力 = 霖楓導師最核心嘅技術。一個普通老師見到錯誤答案會話「錯」。你要喺5秒之內睇穿：呢個錯誤係邊種陷阱造成嘅？然後用陷阱語言框架回應。呢個能力區分你同普通補習老師。

錯誤答案 → 陷阱類型速查表

當你見到學生嘅錯誤答案，用以下映射表快速鎖定陷阱類型。記住：一個錯誤可能同時涉及多個陷阱，你要揀最主要嗰個引爆。

陷阱陷阱名稱錯誤答案特徵常見錯誤例子
T1單位與進位 答案數字啱但單位錯；或者答案大咗/細咗10/100/1000倍 問幾多米，答咗厘米數值；公里轉米漏咗×1000；毫升轉升應該÷1000但×1000
T2小數點 小數點位置明顯偏移；答案多咗或少咗小數位 3.5×0.4應該=1.4但答咗14；小數除法移錯位；加減對唔齊小數點
T3運算次序 答案係「從左到右」計出嚟嘅結果（忽略咗先乘除後加減/括號） 2+3×4應該=14但答咗20；有括號但忽略咗；平方/指數冇先處理
T4面積公式混淆 面積答案係正確數值嘅2倍或者冇÷2 三角形面積漏咗÷2；梯形面積漏咗÷2；菱形面積用咗底×高；平行四邊形面積用咗長×闊
T5幾何圖形混淆 用咗完全錯誤嘅公式（似係將圖形A當成圖形B） 菱形當咗正方形計；梯形同平行四邊形公式調轉；圓形面積用咗圓周公式
T6分數陷阱 分數加減冇通分母直接加分子分母；分數乘除時用錯操作 1/2+1/3答咗2/5（分子加分子，分母加分母）；分數除法冇顛倒相乘
T7百分數陷阱 百分數當成普通數字處理；增加/減少百分比混淆基準 八折當成減$8；「比原價多20%」用錯基準；利息/折扣混淆計算方向
T8速率陷阱 速率=距離×時間（而唔係÷）；單位不匹配 km/h同m/s混用冇轉換；來回行程平均速率計錯（唔係簡單平均）
T9方程陷阱 移項符號錯；等號兩邊操作唔平衡 x+3=7 → x=7+3（移項符號反轉）；兩邊應該同時÷2但只除咗一邊
T10統計圖表陷阱 讀錯刻度/圖例；混淆不同數據系列 棒形圖每格=2但當咗每格=1；折線圖睇錯對應月份；圓形圖忽略咗「不是所有項目總和=100%」
🔍 快速診斷三步法：
Step 1 — 睇答案數量級（太大/太細？→ T1/T2）
Step 2 — 睇運算結構（加減乘除用錯？→ T3/T6/T7/T8/T9）
Step 3 — 睇公式選擇（用咗唔啱嘅公式？→ T4/T5/T10）

常見學生犯錯模式及其揭示

一個獨立嘅錯誤係「事件」。重複出現嘅錯誤模式係「診斷信號」。你要睇嘅唔係一個錯誤，而係錯誤背後的思維習慣。

錯誤模式揭示咗咩問題你嘅回應策略
同一題型連續錯3次 概念未理解，唔係大意。佢嘅心智模型有結構性缺陷。 停止練習。返去重新引爆陷阱。用唔同嘅例子再展示一次錯誤vs正確。叫學生教你一次。
識做但成日漏單位 知道方法，但未形成完整答題習慣。考試會俾人扣步驟分。 用「扣分視覺化」：直接喺佢講義上模擬扣分。「呢度——漏單位，SSPA扣1分。你寧願扣呢1分定寫多兩個字？」
文字題識計但列錯式 閱讀理解問題 > 數學問題。佢睇唔明題目嘅「陷阱關鍵詞」。 帶佢圈出題目關鍵詞。訓練：「比你」、「剩返」、「總共」、「平均」→ 每個詞對應咩運算？
MC題成日揀「最似」嘅錯選項 冇驗算習慣。憑感覺揀答案。 強制驗算三步：① 估算（答案大約幾多？）② 代入（將答案塞返入題目）③ 檢查單位
做做下突然空白停咗 遇到冇見過嘅題型，觸發焦慮凍結。唔係唔識，係panic。 唔好催。用降級策略：「呢題難，我哋先做一個簡單版。（出一個冇陷阱嘅同類題）做啱咗？好，而家加返個陷阱——」
前一堂識，呢一堂完全唔記得 記憶未焊接。上一堂嘅洞察階段夠強，但記憶階段唔夠。 用口訣重啟：「記唔記得我哋個口訣？底乘高係——？」等佢接。口訣係記憶嘅觸發器。

陷阱回應標準句式

每次你診斷出陷阱之後，用以下標準句式回應。呢個句式有三個部分：命名陷阱 → 展示錯誤機制 → 連結避免策略。

✅ 標準回應模板：
「你啱啱中咗 [Tx陷阱名稱]！睇下——你係咪 [具體錯誤動作]？呢個陷阱嘅運作方式係 [陷阱機制解釋]。避開佢嘅方法好簡單——每次見到 [觸發條件]，第一件事就係 [檢查動作]。」

實戰例子：

「你啱啱中咗 T4陷阱（面積公式混淆）！睇下——你係咪 用咗底×高，但呢個係三角形，仲要÷2？呢個陷阱嘅運作方式係 你嘅大腦見到『面積』兩個字就自動提取咗最常用嘅長方形公式。避開佢嘅方法好簡單——每次見到 三角形，第一件事就係 喺個腦入面講一次『底乘高除二』。」

何時停止重教 vs 何時記錄後繼續

停止並重教記錄但繼續
學生連續3題同類型都錯 學生做啱咗步驟但計錯數（運算錯誤）
學生嘅錯誤揭示根本概念缺失（例如唔知三角形係咩） 學生識做但速度慢（需要更多練習而唔係重教）
錯誤涉及今日嘅核心KP陷阱 錯誤係之前嘅課題（記錄後安排針對性補練）
🐢Support學生完全跟唔到 🚀Fast學生做錯一題🌳級別（正常，挑戰區應該有挫敗）
⚠️ 關鍵判斷原則：如果呢個錯誤會令學生「下次上堂完全聽唔明下一課」→ 必須而家重教。如果呢個錯誤只係影響呢一題 → 記錄，繼續前進。你嘅時間係有限資源，投資喺最高回報嘅干預上。

1-on-3 動態掌控

1-on-3 課堂最常見嘅失敗模式：一個學生做緊題，另外兩個等緊（然後開始發夢/玩筆/畫公仔）。你呢個結構必須確保三個學生同時參與每一個環節。

「三人同步」技術

課堂環節學生A（正在答題）學生B + C（觀察者）
輪流實戰 大聲講出思考過程，邊做邊解釋 喺講義上寫低：① A有冇中陷阱？邊個？② A嘅做法同你嘅做法一唔一樣？
陷阱引爆 嘗試指出錯誤答案嘅問題 如果A講漏咗，B/C補充。目標：三個人合力拆解一個陷阱。
分層練習 各自做自己難度嘅題目 各自做自己難度嘅題目（本身就係分軌）。但你巡邏時要同每個人都傾兩句。
口訣背誦 全班一齊 全班一齊。最後一輪由最快完成嘅學生帶讀。

注意力分配原則

你嘅注意力係課堂入面最稀缺嘅資源。分配原則：

KP精講時段：70%注意力喺 🐢Support學生（確保佢跟到） → 20% 🚶Standard → 10% 🚀Fast（佢已經明，俾佢自己探索）

分層練習時段：50% 🐢Support → 30% 🚶Standard → 20% 🚀Fast

應用題+出門券：平均分配，因為全部人做同一題。

微競爭技術（Micro-Competition）

競爭可以係強大嘅動力，但用錯會打擊🐢Support學生。霖楓嘅微競爭唔係比「邊個最叻」，而係比「邊個最識避開陷阱」。

陷阱獵人排行榜（每堂更新）

每堂記錄：邊個學生成功避開最多陷阱（唔係做啱最多題）。做啱題但冇識別陷阱 = 普通。做錯題但識講返出嚟自己中咗邊個陷阱 = 加分。

操作方式：白板角落畫三個簡單圖示（火箭/行人/烏龜）。每當有學生成功識別一個陷阱，喺佢圖示旁邊加一粒星。落堂時最多星嘅學生獲得「今日陷阱獵人」稱號。

🏆 關鍵設計：🐢Support學生嘅「成功識別陷阱」標準比🚀Fast學生低。例如🐢只要識講「我中咗T4」就得；🚀要解釋「T4嘅機制係咩」先有星。咁樣確保三個軌道嘅學生都有機會贏。

處理🐢Support學生跟不上

當你見到🐢Support學生開始眼神呆滯、停筆、或者亂咁寫——佢已經「掉線」咗。你呢個時候必須即刻介入，唔可以「等多陣睇下佢會唔會跟返」。

🔴 🐢Support 急救程序：
1. 降級題目：即刻俾一題冇陷阱嘅最基礎版。「我哋先唔理陷阱住，淨係做呢步——」
2. 旁白示範：你做一步，佢跟住做一步。每一步都講出聲。
3. 微型成功：確保佢做啱一題（即使係最簡單嘅），然後大聲肯定：「見到未？你做到㗎！而家我哋加返個小陷阱——」
4. 課後記錄：落堂後記錄佢邊個位跟唔上。如果連續兩堂同一個位跟唔上，需要同家長溝通（唔係投訴，係「我哋發現咗一個學習關卡，需要你同我哋一齊幫佢」）。

處理🚀Fast學生感到沉悶

🚀Fast學生嘅沉悶唔係因為「佢好叻」——而係因為「你冇俾足夠嘅挑戰佢」。一個感到沉悶嘅🚀Fast學生會做三件事：① 騷擾其他學生 ② 覺得補習冇用 → 走 ③ 養成「數學好容易」嘅錯覺 → SSPA真正挑戰出現時崩潰。

🚀Fast 學生挑戰升級策略：
Level 1 — 跳級題目：直接俾🌳/🏔️級別題目（跨陷阱綜合題）
Level 2 — 陷阱設計師：叫佢設計一題陷阱題考你。「你設計一條會令同學中T4+T5雙重陷阱嘅題目。」
Level 3 — 小導師模式：叫佢用自己文字向🚶Standard學生解釋一個陷阱。「你用30秒解釋T4陷阱俾佢聽——講得清楚過我就加分。」
Level 4 — 真題挑戰：俾佢做歷屆SSPA真題（標明邊年邊題）。「呢題2019 SSPA卷一第8題——60%考生錯。你挑戰下？」
💡 🚀Fast 學生心態管理：最危險嘅唔係佢覺得悶，而係佢建立咗「我唔使努力都叻過人」嘅自我形象。呢種學生去到中學遇到真正挑戰時會即刻崩潰。你要確保佢每堂都有至少一題係「做錯咗」嘅——因為咁樣佢先會學識面對失敗。一個未經歷過失敗嘅🚀Fast學生，係一個未準備好嘅學生。

課後家長溝通（2分鐘精準匯報）

家長最關心嘅唔係「今日教咗咩」（佢唔識數學），而係「我小朋友有冇進步」。你呢2分鐘嘅匯報必須傳遞三個訊息：具體進步證據 + 仍然存在嘅陷阱 + 下一步行動。

📋 2分鐘家長匯報模板：
① 正面開場（20秒）：「XXX今日表現唔錯！佢成功避開咗T4陷阱，呢個係上星期佢成日錯嘅。」
② 具體挑戰（30秒）：「不過我哋發現佢喺T7百分數陷阱仲有啲kick住。例如『比原價多20%』同『原價嘅20%』佢會撈亂。呢個係SSPA高頻陷阱。」
③ 行動建議（30秒）：「今個星期嘅功課我特別放咗3題T7練習。你可以叫佢做完之後講返俾你聽，點解呢題要用乘法而唔係加法——如果佢解釋到，就代表佢真係明。」
④ 收尾（10秒）：「下星期我哋會跟進呢個位。有咩問題隨時WhatsApp我。」
家長類型溝通重點避免說法
焦慮型（成日問「點解仲係錯」） 用陷阱數據回應：「佢而家T4陷阱中招率由60%降到20%。仲有20%，我哋下個目標係清零。」 「慢慢嚟啦」/「佢已經好努力」
放手型（唔理小朋友） 俾具體到爆嘅指令：「今個星期淨係做呢3題。做完影相WhatsApp我。」 「得閒同佢溫下書」
比較型（成日講人哋小朋友） 將焦點轉移到個人進步：「我哋唔同其他人比，我哋同佢上個月比。上個月T4 80%錯，而家20%。」 「人哋小朋友係點點點」
直升機型（成日干預） 俾佢一個「任務」滿足佢嘅參與慾：「你幫我一個忙——每次佢做完功課，你淨係問佢一句：『呢題有冇陷阱？』唔使check答案。」 「你唔使理咁多」

導師：「唔緊要。『唔識』唔係問題——『唔試』先係問題。我哋一齊拆解佢。你睇下呢題——第一步你見到咩數字？」 學生：「（可能沉默或者亂指）」 導師：「你見到『三角形』同『底8cm高5cm』。OK，你呢刻個腦入面有冇一個警報響緊？有把聲話『小心——』？」 學生：「……三角形面積？」 導師：「Exactly！你個腦識得響警報㗎！三角形面積條公式係咩？」 學生：「底乘高除二。」 導師：「睇下！你識㗎！你頭先唔係『唔識』，你係俾個題目嚇親咗。個題目寫得複雜，但核心就係你啱啱講嘅嘢。而家做一次俾我睇。」 （等學生完成） 導師：「搞掂！你頭先覺得自己唔識，但其實你個腦一早有答案。下次唔好講『我唔識』，改講『我要拆解佢』。deal？」
💡 設計原理：將「我唔識」重新框架為「我被題目外表嚇親」。重點係幫學生分別「真嘅唔識」（概念缺失）同「假嘅唔識」（焦慮凍結）。大部分情況係後者。透過引導佢講出公式，證明佢「其實識」，重建信心。

場景 2 · 學生重複犯同一個錯誤

同一陷阱，同一堂，中咗三次
02
導師：「停一停。我哋做一個實驗。」 （遮住題目，只露出關鍵部分） 導師：「你而家唔准計數。你淨係答我一條問題：呢題嘅T4陷阱喺邊度？」 學生：「……要÷2？」 導師：「係。你講得出。但你一做題就會忘記。點解？因為你個腦行緊自動導航。自動導航係用舊路線嘅——而你嘅舊路線係『唔÷2』。我哋要鏟走呢條舊路線。」 （喺白板畫一條路徑圖） 導師：「睇下。舊路線：見到面積 → 底×高 → 完成。新路線：見到面積 → 咩圖形？→ 三角形 → 底×高÷2 → 完成。你每次做題之前，大聲講『咩圖形？』三個字。就係呢三個字。嚟，做一次。」 （學生做，今次做啱） 導師：「就係咁。『咩圖形？』三個字就係你嘅手掣。每次拉一拉佢，你就由舊路線跳去新路線。今個星期嘅功課，我要你每一題旁邊都寫低『咩圖形？→ XX形』先開始計。得唔得？」
💡 設計原理：重複犯錯唔係態度問題——係神經迴路問題。舊嘅錯誤路徑太強，新路徑未夠強。解決方法唔係「再解釋一次」，而係插入一個「中斷訊號」（pattern interrupt）——一個簡單嘅動作/字句，強制打斷自動導航，啟動有意識思考。

💡 設計原理：直接鬧「專心啲！」會產生對抗情緒。用遊戲化語言（陷阱偵測器）將「專心」重新包裝成一個好玩嘅挑戰。重點係令學生自己選擇「開返機」，而唔係被迫聽話。眨下眼係一個非語言嘅提醒訊號，唔會打斷課堂節奏。

導師：「（放低筆，降低聲量，放慢速度）XXX，望住我。抖啖氣。」 「你知唔知，每一個我教過嘅學生——包括我自己——都經歷過你呢個時刻。」 「呢個唔係『你唔叻』嘅證據。呢個係『你個腦正在重組緊』嘅訊號。就好似你砌Lego，砌到一半發覺之前砌錯咗，要拆咗一部分再砌過。呢個過程好唔舒服，但係佢代表緊一樣嘢——你正在由『舊方法』升級去『新方法』。」 （攤開雙手） 「今日我哋唔做呢題。我哋返去一個你一定做到嘅位。三角形面積——底乘高除二。你做一次俾我睇。」 （等佢做啱一題最簡單嘅） 「睇下。你做得到。你頭先以為自己做唔到，但你做到咗。個問題唔喺你度——個問題喺條題目度。條題目太複雜，唔係你嘅能力問題。我哋之後會慢慢加上去，但今日，你已經達成咗最重要嘅嘢：你證明咗自己係做到嘅。OK？」
💡 設計原理：學生情緒崩潰時，任何教學都係無效嘅——大腦處於「戰鬥或逃跑」模式，前額葉（負責思考）關閉。第一步必須係情緒調節（抖啖氣、放慢）。第二步係重新框架（呢個唔係失敗，係升級嘅訊號）。第三步係創造一個「保證成功」嘅微型任務，重建信心。最後將問題歸因於「題目太複雜」而非「學生太蠢」，保護自我形象。

陷阱效能指數（Trap Effectiveness Index, TEI）

TEI = 一個衡量「教學介入係咪有效」嘅單一數字。

📐 TEI 計算公式：
TEI = （本週成功避開陷阱次數 - 上週成功避開陷阱次數）÷ 本週陷阱觸發總次數 × 100

解讀：
TEI > 20 = 教學介入非常有效（學生快速進步）
TEI 0~20 = 有效，但速度可以加快
TEI < 0 = 退步中，需要改變教學策略
TEI 長期 < 0（連續3週）= 觸發紅旗，需要同教學主管討論

陷阱數據如何改良講義

陷阱追蹤嘅最終目的：將課堂數據反饋到講義設計。

原則：如果某個陷阱喺多個學生身上持續出現（連續3週TEI<0），代表講義嘅陷阱展示方式需要改良。可能係：陷阱引爆唔夠衝擊力、口訣唔夠易記、分層練習梯度太大。

反饋流程：每月整理陷阱數據 → 標出「頑固陷阱」（>50%學生TEI<0）→ 向教學主管提交改良建議 → 更新講義相關頁面。

#評估項目評分指引
165分鐘時間紀律 10分=每個時段誤差<1分鐘。5分=偏離但追返。1分=完全失控。
2陷阱引爆衝擊力 10分=每個KP都有陷阱引爆，學生有明顯「啊！」反應。5分=有引爆但反應平淡。1分=冇引爆。

  • 能量曲線：課堂中段（30-45min）有冇能量下跌？點樣可以維持？

同儕觀課表格

每季度一次。邀請另一位霖楓導師觀課（現場或錄影）。觀課者只需回答三條問題：

1. 你見到嘅最強嘅一個教學時刻係咩？（具體描述）
2. 如果呢堂係你教，你會做一樣咩唔同嘅嘢？（建設性建議）
3. 學生離開課室時，你覺得佢哋嘅情緒狀態係咩？（推測）

月度教練對話模板

每月同教學主管進行一次30分鐘對話。結構：

時間主題引導問題
0-5min數據回顧「今個月你嘅自評平均分係幾多？邊項最高/最低？」
5-15min深度案例分析「揀一堂你覺得最難handle嘅課。發生咗咩事？你點樣處理？如果再嚟一次你會點做？」
15-25min技能發展「你想喺邊個範疇突破？我哋可以點樣幫你？」
25-30min下月目標「下個月嘅1個具體目標係咩？（必須可量化）」
◆ ◆ ◆
🎓 最後的話：你而家手上嘅，唔係一本「點樣教數學」嘅手冊。係一套「點樣改變學生同數學嘅關係」嘅操作系統。每一個行過你課室嘅學生，離開嘅時候應該帶走嘅唔係一堆公式——而係一個信念：「數學唔係考我嘅聰明，而係考我嘅警覺。而我，係一個出色嘅陷阱獵人。」

呢個信念，係你種落去嘅。每一堂，每一次陷阱引爆，每一句「你啱啱中咗——但你而家識避喇」，你都喺度改寫一個學生對數學、對學習、對自己能力嘅敘事。
