二、核心知識精講 ＋ 例題練習
知識點一：同分母分數加法 🔴 SSPA
① 黃金規則：分母相同 → 只加分子，分母不變！
② 例：27 + 37 = 2+37 = 57
③ 帶分數加法：整數加整數，分數加分數。分數部分滿咗（分子>=分母）要進位！
④ 絕對不要：連分母都加埋！（27 + 37 ≠ 514 ❌）
知識點二：同分母分數減法 🔴 SSPA
① 分母相同 → 只減分子，分母不變！
② 例：58 - 38 = 5-38 = 28 = 14（約簡後）
③ 帶分數減法：分數部分唔夠減 → 向整數借 1（借 1 當做分母咁多份）
  例：314 - 134 = 254 - 134 = 124 = 112
④ 答案要約簡！（P4只要求簡單約簡，如24→12）
例1
310 + 510 = ？（答案 = 810 = 45）
例2
238 + 158 = ？（38+58=88=1 → 3+1=4 → 答案=4）
例3
216 - 56 = ？（16唔夠減56 → 借1 → 176-56=126=113）
⚠️ 世紀災難：27+37=514 ❌！50%學生會連分母都加埋！分母代表每份有幾大，份數增加但每份大小冇變 → 分母不變！
⚠️ 減法陷阱：58-38唔係等於20（有人減到分母都消失！）。正確= 28=14。

#易錯點（❌ 陷阱）正確做法（✅）
1分母都加埋：25+15=310 ❌分母唔變！只加分子：2+15=35。分母係單位大小！
2答案唔約簡：28 就算要約簡到最簡：28=14。分子分母除公因數
3帶分數加法忘進位：135+245=375 ❌35+45=75=125→3+125=425
4帶分數減法唔識借位：314-134=?14唔夠減34→借1=借44→254-134=124=112
5整數減分數唔識轉：1-35=?1=55→55-35=25
6減法次序搞錯：38-58=28 ❌唔夠減要轉！38-58=負數→呢類題要確認題目係咪合理

🧠 口訣：「同分母加減好簡單，分母不變分子計。帶分整數分開做，分數滿咗要進位；分數唔夠問整數借，借一當做分母份。」
七、解題四步卡
1
驗分母
分母相同？係→可以直接加減分子。唔係→L10會學點處理。
2
計分子
加法：分子相加。減法：分子相減。分母保持不變！
3
約簡
分子分母有公因數→約簡。例：48→12。
4
轉形式
假分數→轉帶分數。帶分數答案→分數部分要最簡。
