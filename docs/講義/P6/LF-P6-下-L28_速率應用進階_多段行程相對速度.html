二、核心知識精講 + 例題練習
知識點一：多段行程與平均速率SSPA 必考 — 世紀災難
平均速率 ≠ 速率的平均！
平均速率 = 總距離 ÷ 總時間
唔係 (速率1 + 速率2) ÷ 2！

例：小明駕車 120 km：頭 60 km 以 60 km/h（用咗1h），後 60 km 以 120 km/h（用咗0.5h）
速率的平均 = (60+120)÷2 = 90 km/h ❌
真正平均速率 = 120÷1.5 = 80 km/h ✅
點解唔同？因為不同速率行駛的時間長短不同，時間長的速率影響更大！
陷阱引爆：世紀災難級
例1：小明駕車從甲地到乙地，全程 240 km。頭 120 km 速率 60 km/h，後 120 km 速率 80 km/h。求全程平均速率。
世紀災難（85% 學生）
平均速率 = (60+80)÷2 = 70 km/h
呢個係速率的平均，唔係平均速率！時間權重不同！
正確解法
t₁ = 120÷60 = 2h
t₂ = 120÷80 = 1.5h
總時間 = 3.5h
平均速率 = 240÷3.5 ≈ 68.6 km/h
頭段用咗較長時間（2h），所以較低速率的影響更大
口訣：「平均速率唔係平均速率！總距離除總時間，唔係兩個速率加埋除二！」
致命陷阱：平均速率 = 總距離 ÷ 總時間。唔好直接將兩個速率平均！呢個錯 = 全題 0 分！

知識點二：相對速度（Relative Speed）SSPA 高頻
兩物體同時移動：
相反方向（面對面/背對背）：
相對速率 = 速率A + 速率B
相遇時間 = 初始距離 ÷ (速率A + 速率B)
相同方向（追及問題）：
相對速率 = |速率A − 速率B|（快的減慢的）
追及時間 = 初始距離 ÷ |速率A − 速率B|
關鍵：要睇清楚係迎面（相加）定同向（相減）！
陷阱引爆：相對速度方向
例2：甲、乙兩車從相距 400 km 的兩地同時出發相向而行。甲車速率 55 km/h，乙車速率 45 km/h。幾小時後相遇？
常見錯誤
相對速率 = 55−45 = 10 km/h
時間 = 400÷10 = 40 h
方向搞錯！面對面應該相加，唔係相減！
正確解法
相對速率 = 55+45 = 100 km/h
相遇時間 = 400÷100 = 4 h
驗算：甲走55×4=220km，乙走45×4=180km。220+180=400 ✓

知識點三：環形跑道問題SSPA 高階
環形跑道（圓形操場）：
同向跑：快者追上慢者 = 快者比慢者多跑一圈
追及時間 = 一圈長度 ÷ 速率差
反向跑：相遇 = 兩人合共跑了一圈
相遇時間 = 一圈長度 ÷ 速率和
關鍵：環形跑道上無起點終點，循環計算

#易錯點正確做法
1平均速率=速率的平均：(60+80)÷2=70總距離÷總時間，時間長的速率影響更大
2相對速度方向混淆：面對面用減法迎面→相加；同向追及→相減（快減慢）
3起步時間不同未處理：直接設相遇時間先到者已走距離要扣減，或分時段處理
4環形跑道當直線：用直線公式計算同向：追及=多跑一圈；反向：相遇=合共一圈
5單位換算出錯：km/h 同 m/s、min 同 h先統一單位：1 m/s = 3.6 km/h，1 h = 60 min
6火車過橋/隧道長度漏計：只計隧道長總距離 = 隧道長 + 火車長度
7來回去程距離不同誤設：假設兩段距離一樣確認題目是否「同一路線」，如不是則各自計算
口訣：
「平均速率總除總，面對面加背對背加，同向追及大減細；環形跑道計圈數，起步時間要扣數！」
