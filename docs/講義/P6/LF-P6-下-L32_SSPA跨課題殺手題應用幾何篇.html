二、殺手題型一：幾何 + 百分數 + 比較（T5+T4混合）
殺手組合A：圓形面積變化 + 百分數增減 SSPA必考壓軸
① 陷阱1（T5圓形公式）：半徑變 → 面積變 r² 倍，不是 r 倍！
② 陷阱2（T4增減方向）：半徑增加20%，新面積 = (1.2r)² × π = 1.44πr² = 原面積的144%
③ 陷阱3（比較基準量）：面積增加了44%，不是20%！
④ 公式：新面積 = 原面積 × (1 + r%)²
🪤 陷阱引爆 — 圓形面積變化殺手題
一個圓的半徑增加了25%。新面積是原面積的多少倍？增加了百分之幾？
❌ 世紀災難
新面積 = 1.25 × 原面積（錯！）
增加了25%（錯！）
60%學生會咁答！面積 ∝ r² → (1.25)² = 1.5625倍，增加了56.25%
✅ 正確解法
(1.25)² = 1.5625倍
增加了 56.25%
記住：面積跟半徑平方成正比！A=πr²，r變k倍 → A變k²倍
⚠️ 呢題係SSPA最經典殺手題之一！面積變化 ≠ 半徑變化%，一定要平方！

🪤 陷阱引爆 — 排水法+速率三合一
一個長方體水箱，底面積500 cm²，原有水深12 cm。放入一塊石頭後，水深升至16.5 cm。（a）求石頭的體積。（b）如果每分鐘注水250 cm³，要令水深由12 cm升至20 cm（不放入石頭），需要多少分鐘？
❌ 常見錯誤
石頭體積 = 500×16.5 = 8250 cm³（用水面高度！）
時間 = 500×20÷250 = 40分鐘（計錯目標水位）
體積用新水位係致命錯誤！正確用：高度差 = 16.5-12 = 4.5cm
✅ 正確解法
(a) V = 500×(16.5-12)=500×4.5=2250 cm³
(b) 水體積=500×(20-12)=4000cm³, 時間=4000÷250=16分鐘
排水法關鍵：用高度差，唔係新高度！注水計目標水量，唔係總水量

🪤 陷阱引爆 — 面積+利潤三合一
一個長方形花園長15m、闊12m，中間有一個半徑3m的圓形水池。餘下部分鋪草皮。（a）草皮面積是多少m²？（b）草皮成本每m² $40，園丁以成本加利潤25%報價。總報價是多少？（取π=3.14）
❌ 常見錯誤
草皮=15×12-3.14×3²=180-28.26=151.74m²
報價=151.74×40×25%=1517.4（錯！）
利潤25%意思是售價=成本×125%，唔係成本×25%！
✅ 正確解法
(a) 草皮=180-28.26=151.74 m²
(b) 成本=151.74×40=$6069.6, 報價=6069.6×1.25=$7587
利潤%基於成本，售價=成本×(1+利潤%)

🪤 陷阱引爆 — 終極四合一殺手題
A、B兩城相距240km。圓形圖顯示：汽車以90km/h完成全程的35%，以72km/h完成45%，餘下路程以60km/h完成。（a）各段路程的距離？（b）全程所需總時間？（c）全程平均速率？（取一位小數）
❌ 世紀災難
平均速率 = (90+72+60)÷3 = 74 km/h（簡單平均！）
各段距離：35%×240=84km, 45%×240=108km, 20%×240=48km
平均速率要用總距離÷總時間！不同速率行不同距離，不能簡單平均
✅ 正確解法
T1=84÷90=0.933h, T2=108÷72=1.5h, T3=48÷60=0.8h
總時間=3.233h, 平均=240÷3.233=74.2 km/h
此題巧合接近簡單平均，但一定要用總距離÷總時間！

#易錯點正確做法
1圓形面積變化% ≠ 半徑變化%：r+20% → A+44%面積跟r²成正比，新面積 = (1+r%)² × 原面積
2排水法用水面新高度：誤用新水位乘底面積用水位差（新水位−舊水位）× 底面積
3利潤%基準量錯誤：利潤% = (售−本)÷本，不是÷售售價 = 成本 × (1+利潤%)
4平均速率用簡單平均：(v1+v2+v3)÷3平均速率 = 總距離 ÷ 總時間
5複合圖形分割/填補法選錯畫圖分析，確認每個子圖形的公式和數據
6單位換算遺漏：m²↔cm², cm³↔L1m²=10000cm², 1L=1000cm³, 1m³=1000L
7跨課題題目只解決第一個問題列出所有涉及的陷阱→逐一擊破→驗算整合
