二、核心知識精講 ＋ 例題練習
知識點一：加減混合運算的順序🔴 必考
① 沒有括號時：由左至右，順序計算
② 有括號時：先算括號裡面，再算外面
③ 加減混合中，順序不同，答案可能不同！
④ 例：(100-50)+30 ≠ 100-(50+30)
例題 1：由左至右
計算：2356 + 1829 - 1650 = ？
步驟1：2356 + 1829 = 4185
步驟2：4185 - 1650 = 2535
🪤 例題 2：陷阱！先加後減 vs 先減後加
比較：A. 500 - 200 + 100   vs   B. 500 + 200 - 100
❌ 小陷阱怪獸話
兩個答案一樣！
錯！A: 500-200+100=400；B: 500+200-100=600。完全唔同！
✅ 真相
A=400, B=600
加減混合運算的順序不能亂調！由左至右係鐵律！
⚠️ 致命陷阱：500-200+100 ≠ 500-300！有同學見到 200+100 就心急加咗先 — 錯！一定要由左至右：先 500-200=300，再加 100=400！

知識點二：括號的力量🔴 必考
① 括號 ( ) 告訴你：先算我裡面的！
② 有括號 → 先算括號內 → 再從左至右計其他
③ 括號改變計算順序，答案也會改變！
④ 例：500 - (200 + 100) = 500 - 300 = 200（跟無括號的 500-200+100 答案不同！）
例題 3
計算：800 - (350 + 280) = ？
步驟1：括號內 350+280=630
步驟2：800-630=170
🪤 例題 4
(4500 - 2800) + (1200 + 800) = ？
步驟1：第一個括號 4500-2800=1700；第二個括號 1200+800=2000
步驟2：1700+2000=3700

知識點三：應用題關鍵詞辨識🔴 必考·最易中伏
常見關鍵詞對照表：
關鍵詞意思運算
一共、總共、合共全部加埋加法 +
剩下、還餘、找回減去一部分減法 -
A比B多/X比Y少比較兩個數量減法 -
再、然後、之後繼續操作視乎上文！
⚠️ 關鍵詞陷阱：「比...多」唔一定用加法！例：「小明比小華多 50 元」→ 如果問小明有幾多 = 小華的錢 + 50（加法）；但如果已知小明，問小華 = 小明的錢 - 50（減法）！
例題 5：兩步應用題
書店原有圖書 3500 本，上午賣出 1250 本，下午又運來 800 本。現在有多少本？
步驟1（圈關鍵詞）：「賣出」= 減、「運來」= 加
步驟2：3500 - 1250 = 2250
步驟3：2250 + 800 = 3050
答：現在有 3050 本。

#易錯點正確做法
1加減混合亂調順序：500-200+100 計成 500-300由左至右，不能跳步！先 500-200=300 再加 100
2有括號時忽略括號括號內的一定要先計！括號 = 最高優先
3「比...多」一定用加法要睇清楚邊個係未知數！已知大的求小的 = 用減法
4應用題漏了中間步驟兩步題 = 兩個算式，要寫清楚每一步
5加減符號睇錯圈起關鍵詞，在旁邊寫 + 或 -，養成習慣
6答案唔合理都照寫用估算檢查：答案是否在合理範圍？
7應用題漏答句文字題必須寫「答：......」否則扣分！
🧠 口訣一：「加減混合左至右，括號優先要記住；關鍵詞要圈出嚟，兩步題分兩次計。」
🧠 口訣二：「比多比少睇清楚，已知大數求小數就用減；已知小數求大數就用加，唔好見到多字就亂加。」
