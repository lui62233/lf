二、核心知識精講 ＋ 例題練習
知識點一：兩位數 ÷ 一位數 直式⭐ 最重要
① 步驟：十位除 → 有剩就拉落個位 → 再除
② 72 ÷ 3：十位7÷3=2...1 → 剩1+個位2=12 → 12÷3=4 → 答案24
③ 驗證：除數 x 商 + 餘數 = 被除數
④ 口訣：「先除十位，有剩加落個位，再除埋佢！」
例題 1：整除（冇餘數）
🍬 96 粒糖，平均分給 4 個小朋友。每人有幾粒？
直式：96 ÷ 4
十位：9÷4=2...1（2x4=8，9-8=1）
剩1+個位6=16，16÷4=4
答案：24 粒（驗證：4x24=96 ✓）
🪤 例題 2：有餘數！（陷阱重點）
85 ÷ 6 = ？
❌ 小陷阱怪獸話
85÷6 = 13...7
7 > 6！餘數大過除數！仲可以除多次！
✅ 正確答案
85÷6 = 14...1
十位8÷6=1...2；25÷6=4...1；餘數1<6 ✓。驗證：6x14+1=84+1=85 ✓
⚠️ 致命陷阱：「餘數 < 除數」係鐵律！永遠檢查：餘數係咪細過除數？如果唔係 → 商可以再加1！
💡 檢查三步曲：① 除數 x 商 = ？② 加返餘數 = ？③ 餘數 < 除數？

知識點二：估商策略🔴 必考
① 估商 = 搵最接近但不超過嘅乘積
② 例：73 ÷ 8 → 8x9=72（最接近73）→ 商=9，餘=1
③ 上調：如果乘積太小 → 商+1
④ 下調：如果乘積太大（超過被除數）→ 商-1
⑤ 祕技：用乘法表逆向諗：「8x?=最接近73但又唔超過73？」
例題 3：估商練習
57 ÷ 7 = ？試估商！
7x8=56（最接近57）→ 商=8，餘=57-56=1
驗證：7x8+1=56+1=57 ✓，1<7 ✓
🎮 估商大作戰！
「估商高手」挑戰：睇到除法即刻估商！
例如見到 59÷7 →即刻諗：7x8=56 → 商8！

知識點三：整除判斷⭐
① 整除 = 冇餘數（餘數 = 0）
② 點知一個數能否被整除？用乘數表檢查！
③ 例：72 可以被 8 整除嗎？8x9=72 → 可以！
④ 例：73 可以被 8 整除嗎？8x9=72, 8x10=80 → 唔可以！餘1
⑤ 能被 2 整除 = 雙數；能被 5 整除 = 個位係 0 或 5；能被 10 整除 = 個位係 0

#易錯點正確做法
1餘數 ≥ 除數：85÷6=13...7餘數必須 < 除數！7>6，商應該係14
2十位除完冇拉落個位十位有剩餘 → 加落個位一齊除
3估商太大或太小用乘法表逆向：除數x?最接近被除數
4忘記驗證除數x商+餘數=被除數，做完必定檢查
5直式對位唔齊商寫喺被除數上面，對齊位數
6整除時餘數寫0但忘記寫整除時餘數=0，要寫清楚
7除法同乘法掉轉：大數÷小數被除數係大數，除數係細數
🧠 口訣一：「先除十位有剩落，加個位數再除過；餘數必定細過除數，鐵律永遠唔會錯！」
🧠 口訣二：「估商要用乘數表，最接近但又唔超；除完用乘嚟驗證，答案正確冇煩惱！」
