二、核心知識精講 ＋ 例題練習
知識點一：L 和 mL 的轉換⭐ 最重要
① 鐵律：1 L = 1000 mL（唔係 100 mL！）
② L → mL：乘 1000。例：2 L = 2 x 1000 = 2000 mL
③ mL → L：除 1000。例：500 mL = 500 ÷ 1000 = 0.5 L
④ 常用對照：250 mL = 0.25 L, 500 mL = 0.5 L, 750 mL = 0.75 L
⑤ 口訣：「升轉毫升乘一千，毫升轉升除一千！」
L（升）=mL（毫升）
1 L=1000 mL
0.5 L=500 mL
0.25 L=250 mL
2 L=2000 mL
1.5 L=1500 mL
例題 1：單位轉換
一瓶汽水 1.25 L = 幾多 mL？
1.25 x 1000 = 1250 mL
驗證：1 L = 1000 mL，0.25 L = 250 mL，加埋 = 1250 mL ✓
🪤 例題 2：陷阱！
3500 mL = 幾多 L？
❌ 小陷阱怪獸話
3500 mL = 35 L
除咗100唔係除1000！1000mL=1L！
✅ 正確答案
3500 mL = 3.5 L
3500 ÷ 1000 = 3.5 L
⚠️ 終極陷阱：1 L = 1000 mL（唔係 100！）。記住：kilo = 1000，所以 1L = 1000mL！
💡 快速轉換法：L → mL = 加三個0（2→2000）；mL → L = 減三個0（3000→3）

知識點二：容器容量 vs 液體體積🔴 必考·P3最難
① 容器容量：個容器「最多可以裝」幾多液體（係一個上限）
② 液體體積：而家容器入面「實際有」幾多液體
③ 關鍵分別：液體體積 ≤ 容器容量（唔會多過容器！）
④ 例子：一個 2 L 的水壺，入面有 1.2 L 水。容量=2L，液體=1.2L
例題 3：容量 vs 液體
一個水桶最多可裝 5 L 水。現在入面有 3.2 L 水。還可以加多少 L 水？
容量 = 5 L，現有液體 = 3.2 L
還可加 = 5 - 3.2 = 1.8 L = 1800 mL

知識點三：容量比較與排序⭐
① 比較容量前 → 先統一單位！全部轉成 mL 或全部轉成 L
② 例：1.2 L vs 850 mL → 1.2 L = 1200 mL > 850 mL
③ 陷阱：唔可以直接比較「1.2」同「850」，單位唔同！
④ 排序：由大至小或由小至大排列
