二、核心知識精講 ＋ 例題練習
知識點一：不進位乘法（拆位法）⭐ 基礎
拆位法 = 把兩位數拆成「幾十」+「幾個」：
① 拆開：23 = 20 + 3
② 分別乘：20 x 4 = 80，3 x 4 = 12
③ 加埋：80 + 12 = 92
直式寫法：個位乘起 → 十位乘 → 加埋答案
例題 1：不進位（拆位法）

80 + 12 = 92

知識點二：進位乘法🔴 必考·最易錯！
① 從個位開始乘：個位 x 一位數
② 如果個位乘積 ≥ 10 → 進位到十位
③ 再乘十位：十位 x 一位數 + 進位
④ 記住：進位要寫細字，加埋落十位嘅乘積！

🪤 例題 4：忘記加進位（致命陷阱！）
57 x 6 = ？
❌ 小陷阱怪獸話
57x6=302
個位7x6=42→寫2進4；十位5x6=30，但忘記加進位4！寫咗30+2=302？
✅ 正確答案
57x6=342
個位7x6=42→寫2進4；十位5x6=30 +4=34→342
⚠️ 致命陷阱：十位乘完之後，一定要加返個位乘積嘅進位！呢個係 P3 學生犯錯最多嘅地方！
💡 必殺技：用「拆位法」驗證直式！57x6 = 50x6 + 7x6 = 300 + 42 = 342。兩種方法答案一致 = 你做啱咗！

#易錯點正確做法
1忘記加進位：十位乘完不加個位的進位進位寫細字在十位上方，乘完十位後加埋佢
2進位寫錯位置個位的進位寫喺十位上方！千祈唔好寫喺答案欄
3直接抄乘法表但忘記位值：26x3 當 2x3+6x3=6+18=618十位的 2 = 20！20x3=60 + 6x3=18 → 78
4直式對位錯誤：乘數寫錯位一位數寫喺個位下面，對齊！
5忘記用拆位法檢查做完直式，用拆位法驗證：十位x + 個位x
6進位數字寫太大，搞亂原數字進位數字寫細啲，喺數字嘅右上方
7估算後唔識比較精確答案應與估算接近，相差太遠 = 計錯
🧠 口訣：「個位乘起進位記，十位乘完加返佢；拆位驗證最穩陣，兩個方法答案齊。」
