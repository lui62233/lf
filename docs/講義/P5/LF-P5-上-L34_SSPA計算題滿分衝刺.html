二、核心知識精講 + 例題練習
知識點一：多位數 + 小數計算陷阱（T1 + T2）🔴 SSPA 必考
多位數計算四鐵律：
① 對齊規則：加法減法 → 相同數位對齊（個位對個位、十分位對十分位）
② 進位標記：加法進位時在上一行標「小1」，減法借位時標「小點」
③ 小數加減：小數點必須對齊！不足位補 0（例如 3.4 + 0.56 → 3.40 + 0.56）
④ 小數乘法：先整數乘 → 數總小數位 → 從右向左點小數點
⑤ 驗證習慣：心算估算（例如 12,345+987,654 ≈ 12萬+99萬 = 約 111萬）
例題 KP1-1：多位數加法（含進位）
計算：2,345,678 + 8,765,432 = ？
❌ 常見錯誤
10,110,110
進位時忘記加到上一級
✅ 正確解法
11,111,110
逐位對齊，每級進位標記，級級檢查
例題 KP1-2：小數乘法陷阱
計算：0.35 × 0.4 = ？
❌ 常見錯誤
0.35 × 0.4 = 1.40
35×4=140，直接寫140忘了小數點
✅ 正確解法
0.35 × 0.4 = 0.140 = 0.14
35×4=140，2+1=3位小數 → 0.140 → 最簡 0.14
🧠 口訣：「加減對齊位，乘先當整數，數完小數位，從右點起走」
⚠️ T2 最高頻錯誤：0.3 × 0.4 = 1.2（忘記小數乘小數會越乘越小！驗證：0.3的0.4倍 < 0.3，答案必 < 0.3）

知識點二：分數計算陷阱總攻（T9）🔴 SSPA 必考
分數計算五步驗證法：
① 看分母：相同？直接加減。不同？先通分
② 找 LCM：用短除法求最小公倍數（不是隨便找一個公倍數！）
③ 通分：分子 = 原分子 × (LCM ÷ 原分母) —— 必須同步擴大
④ 計算：分母不變，分子相加減
⑤ 約簡：找分子分母的 HCF → 約至最簡 → 假分數化帶分數

例題 KP2-1：異分母加法
38 + 512 = ？
❌ 常見錯誤
820
分子+分子，分母+分母：3+5=8, 8+12=20
✅ 正確解法
1924
LCM(8,12)=24 → 924+1024=1924
例題 KP2-2：含帶分數的減法
213 − 112 = ？
❌ 常見錯誤
2−1=1, 13−12=16 → 116
分別減整數和分數，但 13−12 應該是 26−36=−16
✅ 正確解法
56
轉假分數：73−32 → LCM(3,2)=6 → 146−96=56
🧠 口訣：「帶分先轉假，通分搵 LCM，分子跟住乘，答案必定約」

知識點三：綜合計算策略 — 驗證 ＝ 滿分 🔴 SSPA
計算題滿分策略（不是算完就算，要驗證！）：
① 估算驗證法：心算大約值（例如 2.98 × 4.1 ≈ 3×4 = 12，答案應接近 12）
② 逆運算驗證：加法用減法驗、乘法用除法驗
③ 單位檢查：答案的單位對嗎？小數點位置合理嗎？
④ 尾數檢查：只看個位數／最後一位，快速驗證
⑤ 特殊值警惕：×0.5 = ÷2，÷0.5 = ×2，÷0.1 = ×10
例題 KP3-1：估算驗證
計算 198 × 52 = ？先用估算判斷以下哪個答案合理：A. 1,029 B. 10,296 C. 102,960
❌ 不估算就選
隨便選 A 或 C
沒有估算習慣 = 送分給陷阱
✅ 估算驗證
B. 10,296
198≈200, 200×50=10,000 → 唯有 B 接近
例題 KP3-2：分數與小數混合
計算 12 + 0.25 = ？（統一化成小數或分數）
❌ 錯誤
直接加：12+0.25=0.50.25=?
分數和小數不能直接加！
✅ 正確
0.5 + 0.25 = 0.75 = 34
統一化成小數：12=0.5, 0.5+0.25=0.75
⚠️ 呈分試卷一計算題 35% 失分來自「沒有驗證」而非「不會算」。養成估算習慣！

呈分試計算題滿分心法 🔴 必讀
三不做原則：
① 不對齊不做：多位數和小數計算，位數未對齊絕不開始算
② 不先通分不做：異分母分數，未找到 LCM 絕不開始加減
③ 不驗證不交卷：每題心算估算一遍，答案不合理立即重算

呈分試策略：計算題佔卷一 40-50%，這是最容易拿滿分的部分！因為有標準答案，不是開放題。把計算題做到「零失誤」= 呈分試成功的一半。
🧠 計算滿分口訣：「對齊位、通分先、乘後點、約簡尾、估算驗」
