二、核心知識精講 ＋ 例題練習
知識點一：數字組合心算術⭐ 最重要
① 湊十法：見到可以加埋=10嘅數字，先組合！
   例：7+8+3+2 = (7+3)+(8+2) = 10+10 = 20
② 補數法：用最接近嘅整數，再加減返
   例：99+56 = 100+56-1 = 155
③ 拆數法：將數字拆開方便計算
   例：18x5 = (10+8)x5 = 50+40 = 90

知識點二：估算的三大法則🔴 必考
① 法則一：四捨五入到最接近嘅整十/整百
   例：43 → 40（捨），47 → 50（入），152 → 150 或 200
② 法則二：用簡單數字替代
   例：398 x 3 ≈ 400 x 3 = 1200
③ 法則三：估算結果要合理（唔可以同真實答案差太遠！）
   例：198+203 ≈ 200+200 = 400（真實=401，好接近！）
例題 2：估算練習
估算：287 + 314 + 195 大約係幾多？
287 ≈ 300（四捨五入）
314 ≈ 300（四捨五入）
195 ≈ 200（四捨五入）
估算：300+300+200 = 800
真實答案：287+314+195 = 796 → 估算好接近！
🪤 例題 3：估算 ≠ 亂估
小明估算 47 x 9 = 47 x 10 = 470。呢個估算好唔好？
⚠️ 唔夠精準
47x9 ≈ 47x10 = 470
將9變成10，多咗47！真正答案係423，差好遠！
✅ 更好嘅估算
47x9 ≈ 50x9 = 450
將47四捨五入到50，50x9=450。真實=423，接近好多！
⚠️ 估算關鍵：四捨五入後嘅數字要容易計！通常四捨五入到最接近嘅整十或整百！
💡 估算三步：先四捨五入 → 用心算快計 → 檢查估算結果是否合理
