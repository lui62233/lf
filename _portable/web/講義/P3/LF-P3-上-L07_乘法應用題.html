二、核心知識精講
知識點一：乘法關鍵詞大全🔴 必考
關鍵詞意思例子
倍A 的 B 倍 = A x B5 的 3 倍 = 5 x 3 = 15
每人 / 每組 / 每包每份 x 份數每人 4 粒 x 8 人 = 32 粒
共有 / 一共全部加埋（可能係乘）「每盒 6 個，5 盒共有？」→ 6x5
A 比 B 多 X 倍A = B + (B x X)比 5 多 3 倍 = 5 + 5x3 = 20
例題 1：基本乘法應用
🍬 每包糖有 8 粒，買 6 包共有多少粒？
關鍵詞：「每包」= 每份數量、「共有」= 總數
算式：8 x 6 = 48 粒
答：共有 48 粒糖。
🪤 例題 2：「倍」的陷阱！
小美有 7 張貼紙，小玲的貼紙是小美的 4 倍。小玲有多少張？
❌ 小陷阱怪獸話
7 + 4 = 11 張
見到「倍」就用加？錯！「倍」一定係乘法！
✅ 正確答案
7 x 4 = 28 張
「A 是 B 的 X 倍」= B x X = A
⚠️ 致命陷阱 #1：「倍」永遠係乘法，唔係加法！好多同學一見到兩個數字就加埋佢，呢個係 P3 最大嘅陷阱！
⚠️ 致命陷阱 #2：「每人」= 乘法！「每人 3 個，5 個人」→ 3 x 5 = 15，唔係 3 + 5 = 8！

知識點二：「比...多X倍」的應用🔴 進階陷阱
① 「A 比 B 多 3 倍」= A = B + (B x 3) = B x 4
② 「A 是 B 的 3 倍」= A = B x 3
③ 小心分清「是...的倍」vs「比...多倍」！
④ 多 1 倍 = 原來的 2 倍（呢個最易錯！）
例題 3
小強有 15 張貼紙，小明的貼紙比小強多 2 倍。小明有多少張？
比...多 2 倍 = 原來的 (1+2) = 3 倍
算式：15 x 3 = 45 張
（注意：唔係 15 x 2 = 30！）
例題 4：兩步應用題
書架有 4 層，每層放 9 本書。借出 15 本後，還有多少本？
步驟1（乘法）：4 x 9 = 36 本（原有）
步驟2（減法）：36 - 15 = 21 本
答：還有 21 本。

#易錯點正確做法
1「倍」用加法：5的3倍=5+3=8「倍」= 乘法！5的3倍=5x3=15
2「每人」用加法：每人4粒+8人=12「每人」= 乘法！4x8=32
3「比...多X倍」當成「是X倍」多X倍 = 原來的(X+1)倍！
4應用題漏寫單位答案必定要有單位（粒、元、本...）
5兩步題只做一步圈出所有數字，確定每一步的運算
6分不清「共有」是加法定乘法有「每」字 → 乘法；冇「每」字 → 加法
7忘記答句應用題一定要寫「答：......」
