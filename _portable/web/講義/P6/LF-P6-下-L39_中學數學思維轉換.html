二、核心知識精講 ＋ 思維對比
知識點一：算術思維 vs 代數思維 — 不是更難，是不同！🔴 核心轉換
小學算術思維（Arithmetic）中學代數思維（Algebraic）
從已知數出發，一步一步計算用變數（x, y）代表未知數
目標：搵出「答案」= 一個數字目標：建立「關係」= 一條等式
逆向運算：由結果倒推正向建模：設未知數→列方程→解方程
適合簡單、單步驟問題適合複雜、多未知數、抽象問題
例：? + 5 = 12，所以 ? = 7例：x + 5 = 12，所以 x = 7
中學數學 = 同一樣嘢，用更 powerful 嘅工具去做。唔係難咗，係多咗一個工具箱！
例題 1：同一問題，兩種解法（基礎對比）
「小明比小華大 3 歲，兩人年齡之和是 25 歲。小明多少歲？」
小學算術解法（逆向）
兩人加埋 = 25
小明 = 小華 + 3
所以：小華 + (小華+3) = 25
即 2×小華 = 22
小華 = 11，小明 = 14
中學代數解法（正向）
設小華 = x 歲
則小明 = x + 3 歲
方程：x + (x+3) = 25
2x + 3 = 25
2x = 22
x = 11，小明 = 14
⚠️ 🪤 關鍵陷阱：不要試圖用算術方法解決所有問題！中學題目更複雜，算術方法會令你迷失方向。學識「let x = ...」係你嘅新超能力！
🧠 🧠 「算術係倒推，代數係建模。唔好死守算術，學識用變數，複雜問題即刻變簡單！」
知識點二：文字題 → 方程 — 系統化翻譯 5 步法 🔴 中一核心技能
  1. Step 1：讀題，找出未知數 — 題目問什麼，就設什麼為變數
  2. Step 2：設變數（let x = ...） — 用字母代表未知量，寫清楚單位
  3. Step 3：翻譯關係 — 將中文關鍵詞轉為數學符號：
      「比...多」→ + | 「比...少」→ - | 「是...的 n 倍」→ × n | 「共」→ = | 「相差」→ 大減小
  4. Step 4：列方程 — 將所有關係整合為一條等式
  5. Step 5：解方程 + 驗算 — 解出變數後代入原題驗證
例題 2（翻譯示範）
「一個數的 5 倍比它的 3 倍多 14。求這個數。」

翻譯過程：
設這個數 = x
「5倍」→ 5x
「比...多14」→ 5x - 3x = 14
方程：5x - 3x = 14 → 2x = 14 → x = 7
例題 3（兩未知數）
「甲、乙兩數之和為 48，甲是乙的 3 倍。求兩數。」

翻譯過程：
設乙 = x，則甲 = 3x
方程：3x + x = 48 → 4x = 48 → x = 12
所以乙 = 12，甲 = 36

知識點三：用變數概括規律 — 從「計出結果」到「寫出公式」🟡 中一進階
  1. 數字規律：第 n 項 = 用 n 表達的公式（例如：2, 5, 8, 11... → 第 n 項 = 3n - 1）
  2. 圖形規律：第 n 個圖形的火柴數 / 方格數 → 用 n 表示
  3. 函數概念：y = f(x)，輸入 x 就得到對應的 y
  4. 應用：計算任意項而無需逐個列出（例如直接計第 100 項）
例題 4（數字規律）
數列：4, 7, 10, 13, 16, ...
(a) 寫出第 n 項的公式。
(b) 求第 50 項的值。

解：公差 = 3 → 第 n 項 = 3n + 1
第 50 項 = 3×50 + 1 = 151
例題 5（圖形規律）
第 1 個圖案用 4 支火柴，第 2 個用 7 支，第 3 個用 10 支...
(a) 第 n 個圖案用多少支火柴？
(b) 第 20 個圖案用多少支？

解：火柴數 = 3n + 1（因為每次加 3 支）
第 20 個 = 3×20 + 1 = 61 支

知識點四：算術 vs 代數 — 什麼時候該用哪個？🟡 策略選擇
情況推薦方法原因
一步計算（例如：50 的 30% = ?）算術 ✔直接、快速
兩個未知數有關聯代數 ✔一個變數可表達另一個
逆向運算（知道結果找原因）代數 ✔方程直接建模更清晰
規律概括 / 求第 n 項代數 ✔變數才能表達任意項
幾何面積 / 體積兩者皆可視乎複雜程度
關鍵轉念：中學數學不是「變難」，而是問題變得更多元化。你需要的是一個更大的工具箱，而不是丟棄舊的工具！
🧠 🧠 「簡單心算夠用，複雜方程幫手。兩個方法都識，考試無有怕！」
