二、核心知識精講 + 例題練習
知識點一：年齡問題（Age Problems）SSPA 必考
年齡問題關鍵法則：
1. 年齡差永遠不變！爸爸比小明大 28 歲 → 永遠大 28 歲
2. 「n 年後」→ 所有年齡 +n；「n 年前」→ 所有年齡 −n
3. 設現在年齡為 x，然後表達「n 年後/前」的年齡
終極陷阱：「3年後爸爸年齡是小明的2倍」
正確方程：(爸爸+3) = 2 × (小明+3)，唔係只乘小明！
陷阱引爆：年齡問題
例1：爸爸今年 36 歲，小明今年 10 歲。幾年後爸爸的年齡是小明的 2 倍？
常見錯誤（60% 學生）
設 x 年後：36 = 2(10 + x)
36 = 20 + 2x → x = 8
爸爸都要加 x！係 (36+x) = 2(10+x)！
正確解法
(36 + x) = 2(10 + x)
36 + x = 20 + 2x
36 − 20 = 2x − x
x = 16 年後
驗算：16年後爸爸52歲，小明26歲。52=2×26 ✓
口訣：「n年後人人都加n，n年前人人都減n；年齡差永遠不變，設未知數要小心！」
致命陷阱：n年後 = 所有人的年齡都要 +n！唔好只加一邊！

知識點二：連續數字問題（Consecutive Numbers）SSPA高頻
連續整數：設第一個為 x，則為 x, x+1, x+2, ...
連續偶數/奇數：設第一個為 x，則為 x, x+2, x+4, ...
數字位值問題（兩位數）：
設十位數字為 x，個位數字為 y
該數 = 10x + y（唔係 x + y！）
終極陷阱：「一個兩位數，十位數字是個位的2倍」
設個位 = x，十位 = 2x → 該數 = 10(2x) + x = 21x
陷阱引爆：數字位值
例2：一個兩位數，十位數字是個位數字的 2 倍。如果把數字對調，新數比原數小 18。求原數。
致命錯誤
設原數十位=2x, 個位=x
原數=2x+x=3x, 新數=x+2x=3x
3x−3x=0 → 無解???
原數 = 10(2x)+x = 21x！唔係 2x+x！
正確解法
原數 = 10(2x)+x = 21x
新數 = 10x+2x = 12x
21x − 12x = 18 → 9x=18 → x=2
原數 = 21×2 = 42
驗算：42→對調→24。42−24=18 ✓

#易錯點正確做法
1年齡問題只加一邊：n年後只加年輕人，忘記加老年人n年後所有年齡 +n，n年前所有 −n
2數字位值當加法：十位x個位y→原數=x+y原數=10x+y，十位要×10
3連續數字設定錯：偶數設x, x+1, x+2連續偶數/奇數設 x, x+2, x+4
4設未知數不清晰：同一字母代表兩個不同數量只設一個未知數，其他用關係表達
5幾何公式記錯：周界面積調轉或漏除2記熟所有幾何公式再寫方程
6方程解完唔驗算：計完就當啱代入原題目驗證每個條件
7答句不完整：只寫數字不寫單位或漏答部分完整答句包括所有未知數值及單位
口訣：
「設清未知數，關係寫清楚；年齡差永恆，n年人人加；兩位數=十乘十加個；計完記住驗算！」
