二、核心知識精講
知識點一：不等式符號與基本解法
① 四大符號：<（小於）、>（大於）、≤（小於或等於）、≥（大於或等於）
② 基本解法 = 方程解法：x+3 < 8 → x < 5（移項變號同樣適用）
③ 數線表示：x > 4 → 用空心圓 ○ 在4的位置，向右畫箭咀
④ x ≥ 4：用實心圓 ●，代表「4都係答案」
⑤ 留意：不等式答案係一個範圍，唔係一個數！x < 5 代表 x 可以是 4, 3, 0, −100...
🪤 陷阱引爆 — 不等式乘除負數必須反號！
解不等式：−2x > 6
❌ 致命錯誤（80%人中招！）
−2x > 6 → x > −3
兩邊除以−2時冇反轉不等式符號！呢個係不等式最陰濕嘅陷阱
✅ 正確解法
−2x > 6 → x < −3
乘/除負數 → > 變 <，< 變 >（符號反轉！）
⚠️ 不等式終極陷阱：乘或除負數時，不等式符號必須反轉！呢個係全中一最多人錯嘅位，記住呢個規則你已經贏咗80%嘅同學。
口訣：「不等式照方程咁解，但乘除負數要反胎；大於變小於，小於變大於，記住呢個唔會敗。」

知識點二：直角坐標平面 — 象限、描點、讀點 中一核心
① 坐標系組成：x軸（橫）、y軸（縱）、原點O(0,0)
② 四個象限：I (+,+)   II (−,+)   III (−,−)   IV (+,−)
③ 點的表達：(x, y) — x坐標在前，y坐標在後（次序不可調轉！）
④ (3, 5) ≠ (5, 3)！兩個係完全唔同嘅點！
⑤ 描點方法：由原點出發 → 沿x軸移x格 → 再沿y方向移y格
🪤 陷阱引爆 — (x,y) vs (y,x)
點A(2, 5)和點B(5, 2)是否同一點？
❌ 方向盲錯誤
(2,5) = (5,2) （同一點）
x坐標和y坐標次序不能調！(2,5)係x=2,y=5; (5,2)係x=5,y=2，完全唔同位置
✅ 正確答案
不是同一點！(2,5)在第一象限偏左上，(5,2)在第一象限偏右下
記住：(x,y) = (向右移, 向上移)。x係橫，y係縱，次序絕對不能亂

知識點三：直線圖像 y = mx + c 入門 中一核心
① y = mx + c 是一條直線的方程
② m = 斜率（斜度），c = y軸截距（直線與y軸相交的y值）
③ 畫直線方法：代入兩個不同x值 → 計出y → 描兩點 → 連線
④ 例：y = 2x + 1，當x=0, y=1; 當x=2, y=5。描(0,1)和(2,5)，連線
⑤ 這是中一最重要的圖像基礎，中二三四都要用！
例題示範
畫出直線 y = 3x − 2 的圖像。
步驟：
當 x = 0：y = 3(0) − 2 = −2 → 點(0, −2)
當 x = 1：y = 3(1) − 2 = 1 → 點(1, 1)
當 x = 2：y = 3(2) − 2 = 4 → 點(2, 4)
描出這三個點 → 用直尺連成一條直線

#易錯點正確做法
1不等式乘除負數不反號：−2x>6 → x>−3乘/除負數 → 反轉符號：−2x>6 → x<−3
2(x,y)次序調轉：(2,5)當成(5,2)x坐標永遠在前（橫向），y坐標永遠在後（縱向）
3象限符號記錯：以為II係(+,−)I(+,+) II(−,+) III(−,−) IV(+,−)
4直線代入x計y時計錯負數小心負號：y=−x+4, x=3 → y=−3+4=1
5不等式答案寫成一個數：x>5寫成x=5不等式答案係範圍，不是單一數值
6數線上空心和實心圓混淆○ 不含（<或>）、● 包含（≤或≥）
7坐標軸上點的象限判斷：(0,3)唔屬於任何象限在軸上的點不屬於任何象限（x=0或y=0）
口訣：「不等式乘除負數符號反轉，(x,y)次序一定橫先縱後；象限記口訣：一正正 二負正 三負負 四正負。」
