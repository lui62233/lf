二、核心知識精講 ＋ 例題練習
知識點一：四則運算次序 — 「先乘除、後加減」 🔴 SSPA
① 基本規則：先乘除 → 後加減。如果有括號 → 括號最優先
② 同級（+/- 或 ×/÷）：由左至右逐個計
③ 例：5 + 3 × 4 = 5 + 12 = 17（NOT 8×4=32！）
④ 例：20 ÷ 4 × 2 = 5 × 2 = 10（由左至右，NOT 20÷8！）

知識點二：括號的威力 — 改變運算次序 🔴 SSPA 必考
① 括號內的先計：無論括號外面係加減定乘除，括號永遠優先
② 多層括號：( ) 最內 → [ ] 中層 → 最外層
③ 括號內都要跟乘除優先：(2 + 3 × 4) = (2 + 12) = 14
④ 重點：加括號同唔加括號，答案可以完全唔同！

🪤 世紀陷阱：括號前係減號 → 裡面全部變號！
❌ 錯：20 - (8 + 3) = 20 - 8 + 3 = 15（❌❌❌ 70% 學生咁錯！）
✅ 對：20 - (8 + 3) = 20 - 11 = 9（先計括號內！）

如果一定要去括號：20 - (8 + 3) = 20 - 8 - 3 = 9
規則：括號前係減號，去括號時裡面嘅 + 變 -，- 變 +
例：15 - (10 - 3) = 15 - 7 = 8。或：15 - 10 + 3 = 8
知識點三：去括號變號規則 🔴 SSPA 必考！
① 括號前係 "+"：直接去括號，裡面符號不變
  a + (b + c) = a + b + c
  a + (b - c) = a + b - c
② 括號前係 "-"：去括號時，裡面每個符號都要變！
  a - (b + c) = a - b - c（+ 變 -）
  a - (b - c) = a - b + c（- 變 +）
③ P4 安全做法：先計括號內的數字，再用外面嘅運算符號！

例5

知識點四：含乘除的混合運算 🟡 SSPA
① (a + b) × c ≠ a × c + b（❌ 常見錯誤！）→ (a + b) × c = a×c + b×c
② 分配律：這是 P5 內容，P4 只需計括號內先
③ 含乘除的變號：只影響加減號，乘除號不變
  a - (b × c) = a - b×c（括號內先計）

#易錯點（❌ 陷阱）正確做法（✅）
1次序錯：先加減後乘除先乘除、後加減。括號永遠最優先
2減號後去括號不變號括號前係減號→裡面加變減、減變加！
3同級運算唔由左至右加減同級→由左至右。乘除同級→由左至右
4以為減號可以「結合」：30-12+8→30-2030-12+8=18+8=26。除非有括號：(12+8)先計
5乘除同級時右至左：24÷4×2=24÷8=3❌由左至右：24÷4×2=6×2=12
6忘記括號內都要跟乘除優先(2+3×4)=(2+12)=14，唔係(5×4)=20
7文字題綜合算式漏括號「剩下的平分」→ (總數-用咗) ÷ 人數
🧠 口訣：「先乘除後加減，有括號就最先。減號後面去括號，加變減嚟減變加。同級由左到右做，一步一步唔好跳。」
