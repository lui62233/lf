# 霖楓學苑 · LF Academy — VOICE PROFILE
> 提取自：白皮書 v2.1、銷售手冊、冷啟動手冊、60堂 P5/P6 講義、市場研究報告
> 日期：2026-05-22

## 核心定位語
**「不教數學，教避開陷阱。」**
— 所有輸出的統一錨點。否定式開場 + 翻轉預期 = 記憶點。

## 語調特徵

### 節奏與句長
- **短句為主**（8-15字），長句僅用於技術解釋
- 先給結論，再給原因。例：「呢題 60% 學生會錯。因為…」
- 數字先行：具體百分比、分數、年級永遠放在句首建立權威

### 詞彙選擇
- **香港粵語口語融入書面語**：啱、唔、嘅、俾、睇、佢、嘢、點解
- 禁用大陸用語（「屏幕」「視頻」「質量」→「螢幕」「影片」「質素」）
- 禁用過度正式詞（「建議貴家長」→「提提你」）
- 考試術語精準：呈分試、SSPA、卷一、卷二、MC、長答、步驟分

### 陷阱語言系統（專有）
- 🪤 = 陷阱題標記
- ⚠️ = 高頻錯誤警告
- 🔴 = SSPA 必考/最高頻
- 🟡 = SSPA 中高頻
- 🌱🌿🌳🏔️ = 四級難度標記
- 「致命陷阱」「世紀災難」「必死題型」= 極端錯誤後果
- 「陷阱引爆」= 先展示錯誤答案再揭示正確解法

### 權威建立方式
- 用數據建立可信度：「近5年SSPA真題中出現率>80%」
- 用具體錯誤率建立緊迫感：「60%學生會直接用新水位×底面積」
- 用「考試扣分」建立動機：「呢個錯誤 = 扣 3 分 = 跌一個 Grade」
- 引用教材對照：「對應《數學新思維》5下B 單元十」

### 絕對不做
- ❌ 不說「應該」、「或許」、「可能」→ 說「一定」、「保證」、「必定」
- ❌ 不用空洞形容詞（「非常好」、「很重要」）→ 用數字替代
- ❌ 不寫長段落解釋 → 用表格、列表、口訣
- ❌ 不用「學習建議」、「溫馨提示」→ 用「陷阱警告」、「扣分點」
- ❌ 不假設學生/家長聽得懂抽象概念 → 每個概念配具體例子

### 口訣系統
- 每堂必有 1-2 個 🧠 口訣
- 格式：7-14字，押韻或節奏感強，粵語發音優先
- 例：「底乘高係平行四邊，底乘高除二係三角」
- 例：「先睇圖例辨花紋，再搵月份對位置，水平拉線讀刻度」
- 例：「排水法求體積，唔係求總水體積！底面積乘水位差，升幾多乘幾多！」

### 與家長溝通語調
- 直接、務實、不賣關子
- 「你小朋友目前數學大概咩程度？」（不是「請問貴子弟…」）
- 數據先行：「模擬卷得分 72/100 = B grade，呈分試目標 85+ = A grade」
- 行動導向：「建議每週做 3 題陷阱題，重點攻 T4 面積陷阱」

### 與學生溝通語調（講義內）
- 簡潔指令式：「計！」、「畫出黎！」、「圈出關鍵詞！」
- 鼓勵式警告：「呢題好多人錯，但你唔會，因為你識睇陷阱」
- 分層標記：🚀Fast / 🚶Standard / 🐢Support
