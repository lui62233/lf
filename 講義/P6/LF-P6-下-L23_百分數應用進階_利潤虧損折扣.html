二、核心知識精講 + 例題練習
知識點一：利潤與虧損百分比SSPA 必考
核心公式（必定要背）：
利潤 = 售價(SP) − 成本(CP)
虧損 = 成本(CP) − 售價(SP)
利潤百分比 = (利潤 ÷ 成本) × 100%
虧損百分比 = (虧損 ÷ 成本) × 100%
終極關鍵：利潤%和虧損%的基準永遠是成本(CP)，唔係售價(SP)！
呢個係全港 70% 小六學生都會錯嘅位！
陷阱引爆例題 — 世紀災難級
例1：一件衫成本 $200，售 $280。利潤率是多少？
世紀災難（70% 學生錯）
利潤% = (80÷280)×100%
= 28.6%
用咗售價做分母！利潤率永遠以成本為基準！
正確解法
利潤 = 280−200 = $80
利潤% = 80÷200×100% = 40%
分母 = 成本 $200，唔係售價 $280
口訣：「利潤虧損計百分比，成本放分母永遠啱；售價係分子減成本，記住成本係阿媽！」
致命陷阱：利潤% = 利潤÷成本×100%，唔係 ÷售價！呢個錯 = 全題 0 分！

知識點二：折上折（Multiple Discounts）SSPA 終極陷阱
折上折 ≠ 折扣相加！
例：八折再九折 ≠ 七折（80%+90%-100%=70%? 錯！）
正確：八折再九折 = 原價 × 80% × 90% = 原價 × 72%（即 七二折）
公式：最終售價 = 原價 × (1−d₁) × (1−d₂) × ...
關鍵：每次折扣都是在上次折後價的基礎上再打折！
驗證法：用 $100 測試 → 八折 = $80 → 再九折 = $72 → 總折扣 = 28% off，不是 30% off！
陷阱引爆：折上折
例2：一件外套標價 $800，先八折，再額外九折。最終售價是多少？總折扣是幾多？
常見錯誤（80% 學生）
總折扣 = 20%+10% = 30%
售價 = 800×0.7 = $560
折扣率唔可以就咁相加！每次折扣基準唔同！
正確解法
$800×0.8×0.9 = $800×0.72 = $576
總折扣 = (800−576)÷800 = 28%
乘法關係：80% × 90% = 72%
世紀災難：折扣率不可直接相加！20%+10% ≠ 30%！做一百次錯一百次嘅陷阱！

知識點三：複合百分數變化（Compound Percentage Change）SSPA
連續變化 = 乘法，唔係加法！
先加 a% 再減 b%：最終值 = 原值 × (1+a%) × (1−b%)
關鍵判斷：加 10% 再減 10% ≠ 回復原值！
例：$100 + 10% = $110 → $110 − 10% = $99 → 最終少咗 $1！
順序有影響嗎？無！加 a% 再減 b% = 減 b% 再加 a%（乘法交換律）
但最終結果 ≠ 原值 × (a% − b%)！
例3：一件衫的成本是 $300。老闆先加價 40% 做標價，再八折賣出。老闆最終賺定蝕？賺/蝕多少？利潤率是多少？

#易錯點正確做法
1利潤%以售價做分母：利潤÷售價×100%利潤% = 利潤÷成本×100%，分母永遠是成本
2折上折直接相加：八折+九折=七折乘法：0.8×0.9=0.72=七二折
3連續增減直接抵消：+10%再−10%=冇變1.1×0.9=0.99，最終少咗1%
4標價和成本混淆：以標價計算利潤%利潤率永遠以成本為基準
5求原價時方向錯：減30%後售$140，原價=140×1.3原價=140÷0.7（因為是減完之後的70%）
6減價幅度=利潤率減少：減20%→利潤率跌20%要重新計算（售價−成本）÷成本
7多件貨品利潤率直接平均：三件利潤率直接加埋÷3要計總利潤÷總成本×100%
口訣：
「利潤虧損睇成本，折上折係乘法唔係加法；加完再減唔還原，小心每一步基準！」
