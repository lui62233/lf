二、核心知識精講
知識點一：負數世界 — 認識數線上的「另一邊」
① 0 不是最小的數：0 的左邊仲有 −1, −2, −3...（負整數）
② 負數比較：越左越細！−5 < −3 < 0 < 2 < 5
③ 負數加法：加正數 = 向右移；加負數 = 向左移
④ 負數減法：減正數 = 向左移；減負數 = 向右移！（−(−3) = +3）
⑤ 雙重負號：兩個負號等於一個正號！−(−x) = +x
🪤 陷阱引爆 — 雙重負號
計算：5 − (−3) = ?
❌ 常見錯誤
5 − (−3) = 5 − 3 = 2
以為見到減號就減，忽略後面括號內的負號！
✅ 正確解法
5 − (−3) = 5 + 3 = 8
減一個負數 = 加其正數。兩個「負」=「正」
⚠️ 世紀災難：−(−5) = −5 係錯嘅！兩個負號抵消 → −(−5) = +5。呢個係全中一最多人錯嘅概念！
口訣：「減負等於加正，兩個負號變一個正。負負得正唔會變，數線向右唔會亂。」
知識點二：代數式化簡 — 同類項合併（Collecting Like Terms）
① 同類項：變數和次方完全相同的項 → 可以合併（係數加減）
② 3x 和 5x 是同類項（都係 x）→ 3x + 5x = 8x
③ 3x 和 3x² 不是同類項（次方不同）→ 不能合併
④ 致命陷阱：拆括號時括號前係減號 → 括號內每一項都要變號！
⑤ 3x − (2x − 5) = 3x − 2x + 5 = x + 5（唔係 x − 5！）
🪤 陷阱引爆 — 拆括號變號
化簡：5x − (3x − 8) = ?
❌ 致命錯誤（60%人中招）
5x − (3x − 8) = 5x − 3x − 8 = 2x − 8
括號前係減號 → 裡面嘅 −8 要變成 +8！
✅ 正確解法
5x − (3x − 8) = 5x − 3x + 8 = 2x + 8
拆括號規則：−(A−B) = −A + B

知識點三：方程進階 — 未知數在兩邊 中一必考
① 目標：將所有含 x 的項移到一邊，常數項移到另一邊
② 天秤原理：等式兩邊同時加/減/乘/除同一個數，等式仍然成立
③ 步驟：（1）展開括號（2）移項（變號！)（3）合併同類項（4）求解
④ 陷阱：移項時忘記變號！x+5=3x−1 → 5+1=3x−x → 6=2x → x=3
🪤 陷阱引爆 — 移項變號
解方程：4x + 3 = 2x + 11
❌ 常見錯誤
4x − 2x = 11 + 3（移項冇變號！）
2x = 14, x = 7
2x由右邊移到左邊要變 −2x；3由左邊移到右邊要變 −3
✅ 正確解法
4x − 2x = 11 − 3
2x = 8, x = 4
移項必變號：過對面（等號）→ 符號反轉
⚠️ 終極陷阱：呢題 x=4 和 x=7 差好遠。移項唔變號 = 成份卷錯晒。記住：「過河要變號」！

知識點四：負數四則運算 — 加減乘除全攻略
加法規則：
① 正 + 正 = 正（正常）  ② 負 + 負 = 負（(−3)+(−5)=−8）
③ 正 + 負：睇邊個絕對值大 → 符號跟大嘅（5+(−8)=−3, 8+(−5)=3）
減法規則：減一個數 = 加它的相反數 → a−b = a+(−b)
④ 3−8 = 3+(−8) = −5   ⑤ 3−(−8) = 3+8 = 11
乘法規則：
⑥ 正×正=正   正×負=負   負×正=負   負×負=正
⑦ (−3)×(−4) = +12（這個最多人不信！但係真嘅）
🪤 陷阱引爆 — 負負得正
計算：(−6) × (−5) = ?

(−6) × (−5) = −30
負×負應該係正！兩個負數相乘一定出正數
✅ 正確答案
(−6) × (−5) = +30
規則：同號相乘得正，異號相乘得負

#易錯點正確做法
1雙重負號：−(−x) 當作 −x−(−x) = +x。兩個負號抵銷
2拆括號不變號：3x−(2x−5) → 3x−2x−5括號前係減 → 裡面全變號 → 3x−2x+5 = x+5
3負負得正唔信邪：(−3)×(−4) 以為 = −12同號相乘得正：(−3)×(−4)=+12
4移項唔變號：4x+3=2x+11 → 4x+2x=11+3移過等號要變號：4x−2x=11−3 → 2x=8 → x=4
5x² 當負數代入出錯：x=−3時x²=−9(−3)²=+9。平方一定係正數（或0）
6數線比較方向錯：以為−3>−1越左越細：−3 < −1 < 0 < 2
7同類項判斷錯誤：3x+2x²=5x³3x和2x²次方不同，不能合併
口訣：「負負得正係鐵律，拆括減號要變齊；過河移項必變號，代數世界唔使驚。」
