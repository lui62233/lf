二、核心知識精講 + 例題練習
知識點一：含括號方程 — 分配律展開SSPA 必考
解含括號方程三步法：
1. 展開括號（用分配律）：a(b + c) = ab + ac
2. 移項（未知數一邊，常數另一邊）
3. 合併同類項 + 求解
終極陷阱：括號前有負號！
−(2x + 3) = −2x − 3（括號內每項都要變號！）
2 − (x − 5) = 2 − x + 5 = 7 − x（減號後拆括號，裡面 −5 變 +5！）
陷阱引爆例題 — 世紀災難
例1：解方程 2(x + 3) = 14
常見錯誤
2x + 3 = 14 → 2x = 11 → x = 5.5
2只乘咗 x，忘記乘 3！分配律：2要乘括號內每一項！
正確解法
2x + 6 = 14 → 2x = 8 → x = 4
驗算：2(4+3) = 2×7 = 14 ✓
陷阱引爆：負號拆括號
例2：解方程 3x − 2(x − 4) = 5
致命錯誤（65% 學生）
3x − 2x − 4 = 5
x − 4 = 5 → x = 9
−2(x−4) = −2x + 8，唔係 −2x − 4！負號要乘埋 −4 → +8！
正確解法
3x − 2x + 8 = 5
x + 8 = 5 → x = −3
驗算：3(−3) − 2(−3−4) = −9 − 2(−7) = −9 + 14 = 5 ✓
口訣：「拆括號要逐項乘，負號拆括號全部要變面；入面正變負、負變正，漏咗即刻計錯數！」
最高頻死穴：−(x−5) = −x+5，唔係 −x−5！括號前有減號 = 裡面全部正負號反轉！

知識點二：分數方程 — LCM 去分母法SSPA 終極
分數方程標準解法（LCM法）：
1. 找出所有分母的 LCM
2. 方程兩邊同時 × LCM（每一項都要乘！）
3. 約簡後變成整數方程
4. 移項求解
終極關鍵：乘 LCM 時每項都要乘！包括冇分數的項！
例：x2 + x3 = 10 → 兩邊 × 6 → 3x + 2x = 60 → 5x = 60 → x = 12
陷阱引爆：LCM乘漏常數項
例3：解方程 x3 + x4 = 7
世紀災難（80% 學生初學時錯）
兩邊 ×12: 4x + 3x = 7
7x = 7 → x = 1
忘記右邊的 7 都要 ×12！7 都要變成 84！
正確解法
兩邊 ×12: 4x + 3x = 84
7x = 84 → x = 12
驗算：12/3 + 12/4 = 4 + 3 = 7 ✓
終極陷阱：乘 LCM 時必須乘方程兩邊的每一項！常數項都要乘！呢個錯一出現全題 0 分！
