二、核心知識精講
知識點一：認識 mL 和 L⭐ 最重要
① mL（毫升） = 好少嘅容量
   一茶匙約 5mL、一盒飲品約 250mL、一支水約 500mL
② L（升/公升） = 較多嘅容量
   一支大汽水約 2L、一個水桶約 10L、一個浴缸約 200L
③ 1L = 1000mL（同 1kg = 1000g 一樣！都係 x1000！）
例題 1：選擇合適單位
一盒維他奶的容量約＿＿ A. 250mL B. 250L
答案：A. 250mL（250L 等於 250 支大汽水咁多！）
知識點二：單位轉換🔴 必考
黃金公式：
① 1 L = 1000 mL
② L → mL：× 1000
③ mL → L：÷ 1000
常用：250mL = 14L | 500mL = 12L | 750mL = 34L
例題 2
2L 350mL = ？mL
2L = 2 x 1000 = 2000 mL
2000 + 350 = 2350 mL
🪤 例題 3：陷阱！
1L 50mL = ？mL
❌ 常見錯誤
1L 50mL = 150mL
忘記 1L=1000mL！直接將 1 同 50 連埋！
✅ 正確答案
1L 50mL = 1050mL
1L=1000mL；1000+50=1050mL
⚠️ 致命陷阱：1L = 1000mL，唔係 100mL！同重量單位 1kg=1000g 記法完全一樣！kg↔g = L↔mL = 都係 x1000！

知識點三：容量比較與加減⭐
① 先轉成相同單位（全轉 mL 或全轉 L）
② 再比較 / 加減
③ mL 加滿 1000 → 進位 1L
例題 4
1L 500mL + 800mL = ？
轉 mL：1500 + 800 = 2300mL = 2L 300mL
例題 5：退位減法
3L - 1L 350mL = ？
轉 mL：3000 - 1350 = 1650mL = 1L 650mL
