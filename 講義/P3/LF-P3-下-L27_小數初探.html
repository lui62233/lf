知識點一：小數 = 分數嘅另一種寫法⭐ 最重要
① 小數點「.」係一個神奇標記！分隔整數同小數部分
② 0.1 = 110（十分之一）→ 小數點後第一位 = 十分位
③ 例：0.3 = 310，0.7 = 710，0.9 = 910
④ 口訣：「小數點後第一位，分母一定係十！」

例題 1：分數轉小數
把 410 寫成小數。
410 = 零點四 = 0.4
（十分之四 → 小數點後寫 4）
🪤 例題 2：小心小數點！
以下邊個寫法正確？A. 0.5 B. 0.05 C. .5
❌ 常見錯誤
0.05
0.05 = 5100，係百分之五，唔係十分之五！
✅ 正確答案
0.5
0.5 = 510。小數點後一個位 = 十分位
⚠️ 致命陷阱：0.5 唔係 0.05！小數點後有幾個位，數值差好遠！0.5 = 510 = 一半，0.05 = 5100 = 好細！
💡 記憶法：「十分位」= 小數點後第一個位。1個位 = 分母10，2個位 = 分母100（P4先學）

知識點二：小數比較 — 邊個大？🔴 必考
① 比較小數：先睇整數部分，整數大 → 成個數大
② 整數相同 → 睇十分位，十分位大 → 個數大
③ 例：0.3 vs 0.7 → 整數都係0，睇十分位：3<7 → 0.3 < 0.7
④ 例：1.2 vs 0.9 → 整數1>0 → 1.2 > 0.9（唔好只睇小數部分！）
例題 3：小數比較
排列（由小到大）：0.6, 0.2, 0.9, 0.4
整數全部係 0，比較十分位：2 < 4 < 6 < 9
所以：0.2 < 0.4 < 0.6 < 0.9
🪤 例題 4：有整數嘅小數
1.3 和 0.9，邊個大？
❌ 小陷阱怪獸話
0.9 > 1.3
9大過3，所以0.9大啲？錯！忽略咗整數部分！
✅ 正確答案
1.3 > 0.9
先睇整數：1 > 0，所以 1.3 一定大過 0.9
⚠️ 陷阱提醒：比較小數要先睇整數部分！唔好俾小數點後面嘅數字呃到！

知識點三：小數同分數嘅關係⭐
① 小數係分數嘅「另一種寫法」
② 十分位 → 分母 = 10（例：0.4 = 410）
③ 常用對照：0.5=12，0.25=14，0.1=110
④ 小數加減 = 同分母分數加減（十分位對十分位）
