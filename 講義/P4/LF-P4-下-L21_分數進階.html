二、核心知識精講 ＋ 例題練習
知識點一：真分數 vs 假分數 vs 帶分數 🔴 SSPA
① 真分數：分子 < 分母。例：38 數值 < 1（唔夠一個完整）
② 假分數：分子 >= 分母。例：85 數值 >= 1（等於或超過一個完整）
③ 帶分數：整數 + 真分數。例：135 = 1 個完整 + 35
④ 三者係同一數量嘅不同寫法！85 = 135
⚠️ 致命陷阱：假分數唔係「錯」嘅分數！分子大過分母係合法嘅寫法。8/5 = 一個完整蛋糕再加半個多啲！
知識點二：假分數 ↔ 帶分數互換 🔴 SSPA 必考
① 假分數 → 帶分數：分子 ÷ 分母 → 商 = 整數部分，餘數 = 新分子，分母不變
  例：114 → 11 ÷ 4 = 2 餘 3 → 234
② 帶分數 → 假分數：整數 × 分母 + 分子 → 新分子，分母不變
  例：234 → 2 × 4 + 3 = 11 → 114
例1
將 73 化為帶分數。
例2
將 325 化為假分數。
⚠️ 常見錯誤：7/3 → 除錯方向寫成 3/7！記住：分子 ÷ 分母（大除細），唔係細除大！
知識點三：擴分與約分 🔴 SSPA
① 擴分：分子和分母同時乘同一個數，數值不變
  例：12 = 1×42×4 = 48
② 約分：分子和分母同時除同一個數，數值不變。約到最簡 = 分子分母再无公因數
  例：812 ÷4÷4 → 23
③ 擴分約分 = 分數的變身術！值一樣，樣唔同！
例3
將 34 擴分，使分母變成 12。
例4
將 69 約至最簡分數。
❌ 常見錯誤
擴分只乘分母唔乘分子
1/2 → 4/2（錯！值變咗）
✅ 正確做法
分子分母同步乘/除
1/2 → 4/8（對！值不變）
🎮 遊戲：分數變身王
老師出一個分數，學生鬥快講出兩個擴分同兩個約分！例：2/3 → 4/6、6/9（擴）；6/9 → 2/3（約）

#易錯點（❌ 陷阱）正確做法（✅）
1假分數當錯誤：分子>分母就以為計錯假分數係合法寫法！8/5 = 一個完整+3/5
2假→帶除錯方向：7/3 用 3÷7分子÷分母（大除細）：7÷3=2餘1 → 2又1/3
3帶→假計錯：2⅗ → 2+3+5=10（❌）2×5+3=13 → 13/5（整數×分母+分子）
4擴分只乘分母：1/2 → 3/2（❌ 值變咗）分子分母同步乘：1/2 → 3/6（✅）
5約分只除分子：8/12 → 4/12（❌）分子分母同步除：8/12 → 2/3（✅）
6沒有約到最簡就停：12/18=6/9（未最簡！）仲可以除3：6/9=2/3（最簡！）
7帶分數比較只睇分數部分：1⅞ vs 2¼先睇整數部分！2 > 1 → 2¼ 一定大啲
🧠 口訣：「分子除分母得商同餘數，商係整餘係子母照舊。擴分約分同步走，分子分母一齊郁，分數數值未曾變。」
🧠 速記：「真小於一，假大於等於一，帶分整數加分數。分子大過分母變帶分，整數乘分母加分子變假分。」
七、解題四步卡
1
睇類型
分子<分母→真分數。分子>=分母→假分數。有整數+分數→帶分數
2
假→帶
分子÷分母。商=整數，餘數=新分子，分母照舊
3
帶→假
整數×分母+分子=新分子，分母照舊。記住：先乘後加！
4
擴/約
分子分母同步乘（擴分）或除（約分）。數值永遠不變！
